# Operator Interface (หน้าจอช่างปฏิบัติงาน)

## คืออะไร?

Operator Interface คือประสบการณ์แยกต่างหากที่ออกแบบสำหรับมือถือสำหรับช่างปฏิบัติงานในหน้างาน ออกแบบให้เรียบง่ายโดยเจตนา — ปุ่มสัมผัสขนาดใหญ่ การนำทางน้อยที่สุด ไม่มีความยุ่งเหยิง ช่างไม่ต้องใช้ชื่อผู้ใช้หรือรหัสผ่าน เข้าสู่ระบบด้วย PIN 4 หลัก

อยู่ที่ `/operator` และเป็นภาษาไทยเท่านั้น

## สองส่วนหลัก

### Operator Home Dashboard (`/operator`)

จุดเข้าหลังล็อกอินด้วย PIN แสดง 6 การ์ดการดำเนินการ:

| การ์ด | ทำอะไร |
|------|--------|
| คิวงาน | ดูขั้นตอนทั้งหมดที่มอบหมายให้ช่างคนนี้ |
| สแกน QR | สแกน Barcode ของ Work Order หรือขั้นตอนเพื่อไปยังขั้นตอนนั้นโดยตรง |
| รายงานจำนวน | บันทึกจำนวนชิ้นส่วนที่เสร็จในเซสชันล่าสุด |
| รายงานผิดปกติ | แจ้งปัญหาที่ไม่คาดคิด (ปัญหาเครื่องจักร, วัสดุบกพร่อง, ปัญหาความปลอดภัย) |
| ข้อมูลกะ | ดูรายละเอียดกะวันนี้และการมอบหมายเครื่องจักร |
| ผลงาน | ดูสถิติผลงานส่วนตัวของกะ/วันปัจจุบัน |

### Operator Panel (แผงช่างปฏิบัติงาน)

หน้าจอดำเนินขั้นตอนแบบละเอียด เข้าถึงได้โดยแตะงานในคิวหรือสแกน QR Code แสดง:

- ชื่อขั้นตอนปัจจุบันและคำแนะนำ
- รายละเอียด Work Order (ชิ้นส่วน, จำนวน, อ้างอิงแบบ)
- สถานะขั้นตอนและตำแหน่งในลำดับ
- ตัวนับเวลาทำงานที่ active

ปุ่มการดำเนินการ:

| ปุ่ม | ทำอะไร |
|-----|--------|
| เริ่ม | เริ่มขั้นตอน — เริ่มนาฬิกาจับเวลาทำงาน |
| เสร็จ | ทำเครื่องหมายขั้นตอนว่าเสร็จ — ทำให้ขั้นตอนถัดไปพร้อมใช้งาน |
| พัก | หยุดขั้นตอนชั่วคราวพร้อมเหตุผล (รอวัสดุ, ปัญหาเครื่องจักร ฯลฯ) |
| รายงานปัญหา | สร้างรายงานผิดปกติโดยไม่หยุดขั้นตอน |
| เปลี่ยนเครื่องจักร | ย้ายขั้นตอนนี้ไปยังเครื่องจักรอื่น (ต้องได้รับการอนุมัติจากผู้วางแผน) |

## ขั้นตอน Outside ใน Operator Panel

เมื่อช่างถึงขั้นตอน Outside Service จะเห็นไอคอนรถบรรทุกและสถานะ OPO ปัจจุบัน ช่างไม่สามารถ Complete ขั้นตอนด้วยตนเองได้ — ผู้วางแผนต้องบันทึกการรับชิ้นส่วนจากซัพก่อน

## การสแกน QR / Barcode

ทุก Work Order และขั้นตอนมี QR Code ช่างสามารถสแกนเพื่อ:
- ไปยังขั้นตอนที่ถูกต้องใน Operator Panel โดยตรง
- หลีกเลี่ยงการนำทางผ่านเมนูบนหน้าจอขนาดเล็ก

หน้าสแกนยังทำงานเป็น Fallback หากช่างไม่มีการมอบหมายคิวงาน — สามารถสแกน Barcode WO ใดก็ได้เพื่อเริ่มทำงาน

## แนวคิดหลัก

| คำศัพท์ | ความหมาย |
|---------|----------|
| PIN login | รหัส 4 หลัก — ไม่ต้องใช้ชื่อผู้ใช้/รหัสผ่าน |
| คิวงาน | รายการขั้นตอนที่มอบหมายให้ช่างคนนี้วันนี้ |
| Run time | นาฬิกา Active ติดตามว่าขั้นตอนดำเนินการมานานแค่ไหน |
| Abnormal Report | แจ้งปัญหาที่ไม่คาดคิดจากหน้างาน |
| Hold | หยุดชั่วคราวบนขั้นตอนพร้อมเหตุผลที่บันทึกไว้ |
