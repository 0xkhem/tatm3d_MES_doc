# 10. ความรู้การตั้งค่า (Setup Knowledge)

## การเข้าถึงความรู้การตั้งค่า

1. แตะ **เพิ่มเติม (More)** ในแถบการดำเนินการด่วนด้านล่าง
2. แตะ **ความรู้การตั้งค่า (Setup Knowledge)**

[image: setup-knowledge-full — ภาพหน้าจอเต็มของมุมมอง Setup Knowledge]

---

## วัตถุประสงค์

มุมมอง Setup Knowledge แสดงข้อมูลอ้างอิงสำหรับงานบนเครื่องจักรของคุณ — เครื่องมือที่ต้องการ, วัสดุที่ใช้ และคำแนะนำการตั้งค่า ช่วยให้คุณเตรียมพร้อมสำหรับงานถัดไปโดยไม่ต้องถามผู้วางแผน

---

## ข้อมูลที่แสดง

สำหรับแต่ละ Work Order ที่กำหนดให้เครื่องจักรของคุณ มุมมองแสดง:

| ส่วน | เนื้อหา |
|---------|---------|
| Work Order | รหัส WO และชื่อผลิตภัณฑ์ |
| Process Steps | รายการขั้นตอนพร้อมคำจำกัดความกระบวนการ |
| Required Tools | คำจำกัดความเครื่องมือที่เชื่อมโยงกับคำจำกัดความกระบวนการ |
| Materials | ข้อกำหนดวัสดุสำหรับ Work Order |

[image: setup-knowledge-details — ภาพใกล้แสดงข้อมูลเครื่องมือและวัสดุสำหรับ Work Order]

---

## ข้อกำหนด

- ต้องมีการกำหนดกะ (ปิดใช้งานโดยไม่มี)
- ต้องมีการเชื่อมต่อเครือข่าย (ปิดใช้งานเมื่อออฟไลน์)
- ข้อมูลมาจากโมดูล Knowledge (คำจำกัดความกระบวนการ, คำจำกัดความเครื่องมือ, วัสดุ)
