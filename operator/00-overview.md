# Operator (พนักงานปฏิบัติการ) — คู่มือผู้ใช้งาน

## ภาพรวม

คู่มือนี้ครอบคลุมทุกงานที่พนักงานปฏิบัติการหน้างาน (Shop Floor Operator) ดำเนินการในระบบ TAM3D MES หน้าจอสำหรับพนักงานปฏิบัติการเป็นมุมมองแบบแยกส่วนที่ออกแบบมาให้ใช้งานง่ายบนมือถือ เน้นการทำงานที่รวดเร็วโดยลดภาระทางความคิดให้น้อยที่สุด

> **สำคัญ:** หน้าจอพนักงานปฏิบัติการแสดงเป็นภาษาไทยเท่านั้น ภาษาจะถูกตั้งค่าเป็นภาษาไทยโดยอัตโนมัติเมื่อเข้าหน้า `/operator` และจะกลับคืนค่าเดิมเมื่อออกจากหน้านี้

### การเข้าถึงหน้าจอพนักงานปฏิบัติการ

เปิดเบราว์เซอร์ไปที่ `/operator` หน้านี้เป็นหน้าแยกต่างหาก ไม่ได้ใช้แถบนำทางด้านข้างหลัก

[image: operator-entry — ภาพหน้าจอการเข้าสู่ระบบด้วย PIN ของพนักงานปฏิบัติการบนอุปกรณ์มือถือ]

---

### ขั้นตอนการทำงานของพนักงานปฏิบัติการ

กะทำงานทั่วไป:

```
1. เข้าสู่ระบบด้วยรหัส PIN (OPR-XXXX)
2. ดูงานปัจจุบันในส่วน Active Task Hero (งานหลักที่กำลังทำ)
3. เริ่มขั้นตอน → ทำงาน → รายงานจำนวน → เสร็จสิ้นขั้นตอน
4. ไปยังงานถัดไปในคิว
5. รายงานปัญหาที่พบ (เครื่องจักร, เครื่องมือ, วัตถุดิบ, คุณภาพ, ความปลอดภัย)
6. ตรวจสอบการแจ้งเตือนสำหรับข้อความแจ้งเตือนต่างๆ
7. เมื่อหมดกะ: หยุดพักและออก (หรือทำงานต่อ OT)
8. เช้าวันถัดไป: กลับมาทำงานต่อจากขั้นตอนที่หยุดพัก
```
