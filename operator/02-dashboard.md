# 2. แดชบอร์ด — หน้าจอหลัก

## เค้าโครงหน้าจอ

หลังเข้าสู่ระบบ พนักงานปฏิบัติการจะเห็นแดชบอร์ดหลัก ซึ่งแบ่งออกเป็น 3 ส่วน:

```
┌─────────────────────────────────┐
│  HEADER (ส่วนหัว)                │  ← ชื่อพนักงาน, เครื่องจักร, เวลากะ
├─────────────────────────────────┤
│  ALERT STRIP (แถบแจ้งเตือน)      │  ← คำเตือนเรื่องการพักงาน/ซ่อมบำรุง
├─────────────────────────────────┤
│                                 │
│  ACTIVE TASK HERO (งานปัจจุบัน)   │  ← งานที่กำลังทำอยู่ (เลื่อนได้)
│                                 │
│  TASK QUEUE PREVIEW (ตัวอย่างคิว) │  ← ปุ่ม "งานทั้งหมด" พร้อมจำนวน
│                                 │
├─────────────────────────────────┤
│  QUICK ACTIONS BAR (แถบดำเนินการด่วน) │  ← แถบนำทางด้านล่าง
└─────────────────────────────────┘
```

[image: dashboard-full — ภาพหน้าจอเต็มของแดชบอร์ดพนักงานปฏิบัติการแสดงทั้ง 3 ส่วนบนอุปกรณ์มือถือ]

---

## ส่วนหัว (Header)

แถบส่วนหัวแสดงข้อมูลดังนี้:

| องค์ประกอบ | คำอธิบาย |
|------------|----------|
| ชื่อพนักงาน | ชื่อของคุณ (เช่น "พี่โบ้") |
| รหัสพนักงาน | รหัสของคุณ (เช่น "OPR-0042") |
| เครื่องจักร (Machine) | รหัสและชื่อเครื่องจักรที่ได้รับมอบหมาย (เช่น "CNC-5A-01 · 5-Axis CNC Mill") |
| กะ (Shift) | ชื่อกะ (เช่น "กะเช้า") |
| เวลาที่เหลือ (Time remaining) | นับถอยหลังจนหมดกะ (เช่น "4h 30m") หรือ "หมดกะแล้ว" |
| ปุ่มออกจากระบบ (Logout) | ไอคอน ← สำหรับออกจากระบบ |

[image: dashboard-header — ภาพขยายแถบส่วนหัวแสดงข้อมูลพนักงาน เครื่องจักร และเวลาที่เหลือ]

---

## แถบแจ้งเตือน (Alert Strip)

หากมีการแจ้งเตือนที่ใช้งานอยู่ จะปรากฏแถบเลื่อนแนวนอนด้านล่างส่วนหัว:

| ประเภทการแจ้งเตือน | สี | ไอคอน | ความหมาย |
|---------------------|-----|-------|----------|
| Maintenance (ซ่อมบำรุง) | ส้ม | ⚠ | เครื่องจักรของคุณอยู่ในสถานะซ่อมบำรุง |
| Hold (พักงาน) | เหลือง | ⏸ | ใบสั่งผลิตบนเครื่องจักรของคุณถูกพักงาน |

[image: dashboard-alert-strip — ภาพหน้าจอแถบแจ้งเตือนแสดงข้อความพักงานและ/หรือคำเตือนซ่อมบำรุง]

---

## แถบดำเนินการด่วน (Quick Actions Bar) ด้านล่าง

แถบนำทางด้านล่างที่ตรึงไว้ มี 5 ปุ่ม:

| ปุ่ม | ไอคอน | คำอธิบาย |
|------|-------|----------|
| สแกน (Scan) | 📱 | เปิดเครื่องสแกนบาร์โค้ด/QR |
| งานทั้งหมด (All Tasks) | 📋 | เปิดคิวงานทั้งหมด (ป้ายแสดงจำนวน) |
| รายงาน (Report) | ⚠ | รายงานปัญหา |
| แจ้งเตือน (Alerts) | 🔔 | ดูการแจ้งเตือน (ป้ายแสดงจำนวนการแจ้งเตือน) |
| เพิ่มเติม (More) | ⋯ | ขยายเมนูเพิ่มเติม: ข้อมูลกะ, ความรู้การตั้งค่า, ผลงาน, รายงานของฉัน |

[image: dashboard-quick-actions — ภาพขยายแถบดำเนินการด่วนด้านล่างแสดงทั้ง 5 ปุ่ม]

### เมนูเพิ่มเติม (More Menu)

การกด "เพิ่มเติม" จะแสดงหน้าต่างซ้อนทับพร้อมตัวเลือกเพิ่มเติม:

| ตัวเลือก | ไอคอน | คำอธิบาย |
|----------|-------|----------|
| ข้อมูลกะ (Shift Info) | 🕐 | ดูรายละเอียดกะ |
| ความรู้การตั้งค่า (Setup Knowledge) | 📖 | ดูคำแนะนำการตั้งค่าเครื่องสำหรับงานปัจจุบัน |
| ผลงาน (Performance) | 📊 | ดูสถิติผลงานประจำกะของวันนี้ |
| รายงานของฉัน (My Reports) | 📄 | ดูรายงานที่คุณส่งแล้ว |

[image: dashboard-more-menu — ภาพหน้าจอหน้าต่างเมนูเพิ่มเติมที่ขยายออก]

## คำเตือนไม่มีกะ (No Shift Warning)

หากคุณไม่มีการกำหนดกะสำหรับวันนี้ จะปรากฏแบนเนอร์เตือนสีเหลือง:

> "ไม่มีกะที่กำหนด"

เมื่อเกิดกรณีนี้:
- คิวงานถูกปิดใช้งาน (ไม่มีเครื่องจักรสำหรับกรองงาน)
- การแจ้งเตือนถูกปิดใช้งาน
- ข้อมูลกะและความรู้การตั้งค่าถูกปิดใช้งาน
- **สแกน** และ **รายงาน** ยังคงใช้งานได้

[image: dashboard-no-shift — ภาพหน้าจอแสดงแบนเนอร์เตือนไม่มีกะ]
