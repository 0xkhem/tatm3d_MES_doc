# 3. งานปัจจุบันและการดำเนินการด่วน (Active Task & Quick Actions)

## Active Task Hero (งานปัจจุบันหลัก)

พื้นที่หลักของแดชบอร์ดจะแสดง **Active Task Hero** ซึ่งเป็นงานปัจจุบันของคุณ นี่คือข้อมูลสำคัญที่สุดบนหน้าจอ

ส่วน Hero แสดงข้อมูลดังนี้:

| องค์ประกอบ | คำอธิบาย |
|------------|----------|
| WO Code (รหัสใบสั่งผลิต) | รหัสใบสั่งผลิต (เช่น "WO-2026-00042") |
| Step name (ชื่อขั้นตอน) | ขั้นตอนกระบวนการปัจจุบัน (เช่น "CNC Machining") |
| Product name (ชื่อผลิตภัณฑ์) | ชิ้นงานที่กำลังผลิต |
| Customer (ลูกค้า) | ชื่อลูกค้า |
| Scheduled time (เวลาที่กำหนด) | เวลาที่กำหนดสำหรับการดำเนินงานนี้ (วันที่ + ชั่วโมง) |
| Priority (ลำดับความสำคัญ) | ตัวบ่งชี้แบบภาพ (เร่งด่วน = แดง, สูง = ส้ม) |
| Action buttons (ปุ่มดำเนินการ) | เริ่มงาน / เสร็จสิ้น / รายงานจำนวน ขึ้นอยู่กับสถานะของขั้นตอน |

## การดำเนินการด่วนจากส่วน Hero

ปุ่มดำเนินการที่แสดงจะแตกต่างกันตามสถานะของขั้นตอนปัจจุบัน:

### เมื่อขั้นตอนอยู่ในสถานะ Pending (รอดำเนินการ — พร้อมเริ่มงาน)

- **เริ่มงาน (Start Step)** — ปุ่มสีเขียว กดเพื่อเริ่มทำงานในขั้นตอนนี้

[image: active-task-start — ภาพหน้าจอแสดงปุ่ม "เริ่มงาน" สีเขียวบนงานที่รอดำเนินการ]

### เมื่อขั้นตอนอยู่ในสถานะ In Progress (กำลังดำเนินการ)

- **รายงานจำนวน (Report Qty)** — ปุ่มแบบเส้นขอบ สำหรับรายงานจำนวนที่เสร็จและจำนวนเสีย
- **เสร็จสิ้น (Complete Step)** — ปุ่มสีน้ำเงิน สำหรับทำเครื่องหมายว่าขั้นตอนเสร็จแล้ว

[image: active-task-in-progress — ภาพหน้าจอแสดงปุ่มรายงานจำนวนและเสร็จสิ้นขั้นตอน]

### การไปยังคิวงานทั้งหมด

ด้านล่างส่วน Hero จะมีปุ่มแสดง:

> **งานทั้งหมด** (All Tasks) — พร้อมป้ายแสดงจำนวนงานที่เหลือ

กดปุ่มนี้เพื่อเปิดคิวงานทั้งหมด

[image: active-task-queue-button — ภาพหน้าจอปุ่ม "งานทั้งหมด" พร้อมป้ายแสดงจำนวนงาน]
