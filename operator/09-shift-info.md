# 9. ข้อมูลกะ (Shift Info)

## การเข้าถึงข้อมูลกะ

1. แตะ **เพิ่มเติม (More)** ในแถบการดำเนินการด่วนด้านล่าง
2. แตะ **ข้อมูลกะ (Shift Info)**

[image: shift-info-full — ภาพหน้าจอเต็มของมุมมอง Shift Info แสดงแถวข้อมูลทั้งหมด]

---

## ข้อมูลที่แสดง

| ฟิลด์ | คำอธิบาย |
|-------|-------------|
| ชื่อพนักงาน | ชื่อของคุณ |
| รหัสพนักงาน | รหัส OPR-XXXX ของคุณ |
| ชื่อกะ | ชื่อกะปัจจุบัน (เช่น "กะเช้า") |
| เวลาเริ่มกะ | เวลาเริ่มกะ (เช่น "08:00") |
| เวลาสิ้นสุดกะ | เวลาสิ้นสุดกะ (เช่น "17:00") |
| เครื่องจักรที่กำหนด | ชื่อและรหัสเครื่องจักร |
| เวลาที่เหลือ | นับถอยหลังจนถึงสิ้นสุดกะ หรือ "หมดกะแล้ว" (สีแดง) |

[image: shift-info-rows — ภาพใกล้ของแถวข้อมูลพร้อมข้อมูลตัวอย่าง]

---

## เวลาที่เหลือ

เวลาที่เหลืออัปเดตทุกนาที เมื่อกะสิ้นสุดแล้วจะแสดงเป็นข้อความสีแดง

> **ต้องมีการกำหนดกะ** มุมมองนี้จะถูกปิดใช้งานถ้าคุณไม่มีการกำหนดกะ
