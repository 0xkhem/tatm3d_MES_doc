# TAM3D MES — คู่มือผู้ใช้งาน

## TAM3D MES คืออะไร?

TAM3D MES คือระบบบริหารจัดการการผลิต (Manufacturing Execution System) ที่ออกแบบมาสำหรับโรงงาน CNC ที่ผลิตชิ้นงานหลากหลายประเภทในปริมาณน้อย (High-Mix Low-Volume หรือ HMLV) ระบบติดตามวงจรชีวิตของคำสั่งซื้อทั้งหมด ตั้งแต่การเสนอราคา การวางแผนการผลิต การดำเนินงานในหน้างาน การตรวจสอบคุณภาพ จนถึงการจัดส่ง

ระบบนี้ออกแบบมาสำหรับโรงงานงานกลึงความแม่นยำที่ผลิตชิ้นส่วน เช่น เรือนมอเตอร์ แผ่นอะแดปเตอร์เกียร์ บล็อกแมนิโฟลด์ไฮดรอลิก และชุดประกอบแบร็กเก็ตสำหรับ OEM ยานยนต์และซัพพลายเออร์ระดับ Tier-1

## แนวคิดสำคัญ

### Work Order (ใบสั่งผลิต)

งานผลิตเดี่ยวสำหรับชิ้นส่วนเฉพาะ จำนวน และเส้นทางกระบวนการ รูปแบบรหัส: `WO-2026-00001`

### Process Route (เส้นทางกระบวนการ)

ลำดับขั้นตอนการผลิตที่ชิ้นงานต้องผ่าน (เช่น Die Casting → Deburring → CNC Milling → Surface Treatment → QC → Packing)

### Process Definition (คำจำกัดความกระบวนการ)

เทมเพลตการดำเนินงานที่ใช้ซ้ำได้ (เช่น "CNC Milling", "Deburring", "Anodizing") แต่ละคำจำกัดความจะเชื่อมโยงกับประเภทเครื่องจักร

### Schedule Assignment (การกำหนดตารางการผลิต)

การดำเนินงาน (ขั้นตอน WO) ที่ถูกกำหนดให้เครื่องจักรเฉพาะเครื่อง ในวันที่และเวลาที่กำหนดบนบอร์ดจัดตาราง

### Outside Processing Order (OPO) (คำสั่งงานภายนอก)

เมื่อขั้นตอนใดต้องการงานจากซัพ (เช่น การอบชุบ, การชุบผิว) OPO จะติดตามการจัดส่งไปและรับกลับจากซัพ

### NCR (Non-Conformance Report) (รายงานความไม่สอดคล้อง)

บันทึกคุณภาพที่สร้างขึ้นเมื่อการตรวจสอบไม่ผ่าน ติดตามข้อบกพร่อง สาเหตุรากฐาน และการตัดสินใจ (ซ่อมแก้ไข, ผลิตใหม่, คัดทิ้ง ฯลฯ)

### Delivery Promise Signal (สัญญาณการส่งมอบ)

ตัวบ่งชี้ความเป็นไปได้แบบเรียลไทม์ (เขียว/เหลือง/แดง) ที่ตรวจสอบกำลังการผลิตของโรงงานเทียบกับวันส่งมอบที่ร้องขอ

## เริ่มต้นใช้งาน

### สำหรับผู้ใช้ใหม่

1. รับข้อมูลเข้าสู่ระบบจากผู้ดูแลระบบ
2. เปิด URL ของแอปในเบราว์เซอร์
3. เข้าสู่ระบบด้วยชื่อผู้ใช้และรหัสผ่าน (หรือรหัส PIN สำหรับช่างปฏิบัติงาน)
4. ค้นหาคู่มือตามบทบาทของคุณด้านบนและเริ่มจากหน้าภาพรวม

### สำหรับช่างปฏิบัติงาน (Operator)

1. ไปที่ `/operator` บนอุปกรณ์มือถือ
2. ป้อนรหัส PIN 4 หลัก
3. เริ่มทำงานตามคิวงานของคุณ

### สำหรับผู้ดูแลระบบที่ตั้งค่าระบบ

ก่อนที่ระบบจะสามารถดำเนินการผลิตได้ ต้องตั้งค่าข้อมูลหลักในโมดูล Knowledge ก่อน:

```
1. Materials (วัสดุ — สิ่งที่คุณผลิต)
2. Process Definitions (คำจำกัดความกระบวนการ เช่น CNC Milling, Deburring)
3. Equipment Types (ประเภทเครื่องจักร)
4. Equipment (เครื่องจักรแต่ละเครื่อง)
5. Process Routes (ลำดับขั้นตอนสำหรับชิ้นงาน)
6. Operators (ช่างปฏิบัติงานพร้อมรหัส PIN)
7. Shifts & Work Calendar (กะทำงานและปฏิทินทำงาน)
8. Customers (ลูกค้า)
```

แดชบอร์ด Knowledge Health ติดตามความสมบูรณ์ของข้อมูลและแนะนำขั้นตอนการตั้งค่า
