# Production (การผลิต)

## คืออะไร?

โมดูล Production ติดตาม Work Order ทุกใบตั้งแต่สร้างจนเสร็จสมบูรณ์ เป็นแกนหลักของระบบ — ทุกสิ่งที่เกิดขึ้นในหน้างานจะสะท้อนอยู่ที่นี่

## เอนทิตีหลัก

```
Sales Order → Project → Work Orders → Steps (ขั้นตอน)
```

**Work Order (WO)** — งานผลิตชิ้นเดียวสำหรับชิ้นส่วนหนึ่งชิ้น จำนวนหนึ่ง และเส้นทางกระบวนการหนึ่งเส้นทาง แต่ละ WO มีขั้นตอนที่ช่างปฏิบัติงานดำเนินการทีละขั้น

รูปแบบรหัส WO: `WO-2026-00001`

**Project** — จัดกลุ่ม Work Order ทั้งหมดที่เป็นของ Sales Order เดียวกัน ติดตามความคืบหน้าการส่งมอบโดยรวมของ WO หลายใบ

รูปแบบรหัส Project: `PRJ-2026-001`

## สถานะ Work Order

| สถานะ | ความหมาย |
|-------|----------|
| Draft | สร้างแล้วแต่ยังไม่ปล่อยลงพื้นที่ผลิต |
| Released | พร้อมสำหรับการจัดตารางและการผลิต |
| In Progress | ขั้นตอนอย่างน้อยหนึ่งขั้นตอนเริ่มแล้ว |
| On Hold | หยุดชั่วคราว — ต้องระบุเหตุผล (ขาดวัสดุ, ปัญหาคุณภาพ ฯลฯ) |
| Completed | ทุกขั้นตอนเสร็จสมบูรณ์ |
| Cancelled | ยกเลิกแล้ว — สถานะสุดท้าย |

## สถานะขั้นตอน

| สถานะ | ความหมาย |
|-------|----------|
| Pending | ยังไม่เริ่ม รอขั้นตอนก่อนหน้า |
| In Progress | กำลังดำเนินการอยู่ |
| Completed | เสร็จแล้ว |
| Skipped | ข้ามไปสำหรับล็อตนี้ |

## ฟีเจอร์อื่นใน Production

**Production Kanban** — บอร์ดการ์ดแสดง WO ที่ active ทั้งหมดจัดกลุ่มตามสถานะ เหมาะสำหรับดูภาพรวมว่างานไหนกำลังเดินหน้าและงานไหนติดขัด

**Production Dashboard** — KPI cards (WO ที่กำลังดำเนินการ, อัตราตรงเวลา, จำนวนเกินกำหนด) พร้อมกราฟการกระจายสถานะและอายุ WO

**Order Traceability** — ติดตาม Sales Order ตั้งแต่ต้นจนจบ: WO ไหนถูกสร้าง, ขั้นตอนไหนเสร็จ, การตรวจสอบไหนดำเนินการ, และมี NCR ถูกสร้างหรือไม่

**Abnormal Reports** — ช่างปฏิบัติงานสามารถแจ้งปัญหาที่ไม่คาดคิด (เครื่องจักรเสีย, วัสดุบกพร่อง, ปัญหาความปลอดภัย) ระหว่างการผลิต รายงานจะมองเห็นได้โดยผู้วางแผนและผู้จัดการ

## ขั้นตอนการทำงานประจำวันทั่วไป

1. ฝ่ายขายสร้าง SO และสร้าง WO
2. ผู้วางแผน Release WO และมอบหมายขั้นตอนให้เครื่องจักรบน Scheduling Board
3. ช่างปฏิบัติงานดำเนินขั้นตอนและรายงานจำนวน
4. ผู้วางแผนติดตามความคืบหน้าบน Dashboard และ Kanban
5. WO ที่เสร็จสมบูรณ์รวมเข้า Project → SO สำเร็จ
