# 2. Inspections — รายการและค้นหา

## การเข้าถึงรายการการตรวจสอบ (Inspection List)

ไปที่ **Quality → Inspections** หรือเข้าเส้นทาง `/quality/inspections`

[image: inspections-list-full — ภาพหน้าจอเต็มของหน้ารายการ Inspections แสดงแถบเครื่องมือและตาราง]

---

## เค้าโครงหน้า (Page Layout)

### แถบเครื่องมือ (Toolbar)

- **ช่องค้นหา** (Search box) (ด้านซ้าย) — กรองตามรหัสการตรวจสอบ, รหัส WO หรือชื่อผลิตภัณฑ์
- **ปุ่ม + Inspection** (ด้านขวา) — สร้างการตรวจสอบใหม่

[image: inspections-toolbar — ภาพใกล้ของแถบเครื่องมือพร้อมช่องค้นหาและปุ่มสร้าง]

### ตาราง Inspections

| คอลัมน์ | คำอธิบาย |
|--------|-------------|
| Inspection No. (เลขที่ตรวจสอบ) | รหัสที่สร้างอัตโนมัติ (แบบ monospace) |
| Work Order (ใบสั่งงาน) | รหัส WO ที่เชื่อมโยง |
| Product (ผลิตภัณฑ์) | ชื่อผลิตภัณฑ์จาก WO ที่เชื่อมโยง |
| Process (กระบวนการ) | ชื่อขั้นตอนกระบวนการ (เช่น "CNC Machining", "Surface Treatment") |
| Date (วันที่) | วันที่ตรวจสอบ |
| Result (ผลลัพธ์) | ป้ายสีแสดงผล: Pass (เขียว), Fail (แดง), Conditional Pass (เหลือง) |
| Inspector (ผู้ตรวจสอบ) | ชื่อผู้ที่ทำการตรวจสอบ |
| Defect Count (จำนวนข้อบกพร่อง) | จำนวนข้อบกพร่องที่พบ (ชิดขวา) |

[image: inspections-table — ภาพหน้าจอของตารางที่มีข้อมูลตัวอย่าง 5-6 แถวแสดงผลลัพธ์ต่างๆ]

---

## การค้นหา (Searching)

พิมพ์ในช่องค้นหาเพื่อกรองตาม:
- รหัสการตรวจสอบ (Inspection code)
- รหัส WO (WO code)
- ชื่อผลิตภัณฑ์ (Product name)

ตารางจะกรองข้อมูลแบบเรียลไทม์

[image: inspections-search — ภาพหน้าจอแสดงผลลัพธ์หลังจากค้นหา]

---

## การแบ่งหน้า (Pagination)

หากมีการตรวจสอบมากกว่า 20 รายการ จะแสดงตัวควบคุมการแบ่งหน้าด้านล่าง:
- ข้อความ "Showing X–Y of Z" (แสดง X–Y จาก Z)
- ปุ่ม Previous (ก่อนหน้า) / Next (ถัดไป)

[image: inspections-pagination — ภาพใกล้ของตัวควบคุมการแบ่งหน้า]

---

## ป้ายผลลัพธ์ (Result Badges)

| ป้าย | สี | ความหมาย |
|-------|-------|---------|
| Pass | พื้นหลังเขียว, ตัวอักษรเขียว | ชิ้นงานเป็นไปตามข้อกำหนดทั้งหมด |
| Fail | พื้นหลังแดง, ตัวอักษรแดง | ชิ้นงานไม่เป็นไปตามข้อกำหนด — อาจต้องสร้าง NCR |
| Conditional Pass | พื้นหลังเหลือง, ตัวอักษรเหลือง | ชิ้นงานได้รับการยอมรับโดยมีค่าเบี่ยงเบนที่บันทึกไว้ |

[image: inspections-result-badges — ภาพใกล้แสดงป้ายทั้งสามประเภทในตาราง]
