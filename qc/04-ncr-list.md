# 4. NCR — รายการและค้นหา

## การเข้าถึงรายการ NCR

ไปที่ **Quality → NCR** หรือเข้าเส้นทาง `/quality/ncr`

[image: ncr-list-full — ภาพหน้าจอเต็มของหน้ารายการ NCR แสดงแถบเครื่องมือพร้อมช่องค้นหา ตัวกรองสถานะ และตาราง]

---

## เค้าโครงหน้า (Page Layout)

### แถบเครื่องมือ (Toolbar)

- **ช่องค้นหา** (Search box) — กรองตามรหัส NCR, รหัส WO หรือชื่อผลิตภัณฑ์
- **ดรอปดาวน์ตัวกรองสถานะ** (Status filter dropdown) — กรองตาม: All (ทั้งหมด), Open (เปิด), In Review (อยู่ระหว่างตรวจสอบ), Resolved (แก้ไขแล้ว), Closed (ปิดแล้ว)
- **ปุ่ม + NCR** — สร้าง NCR ใหม่

[image: ncr-toolbar — ภาพใกล้ของแถบเครื่องมือแสดงช่องค้นหา ตัวกรองสถานะ และปุ่มสร้าง]

### ตาราง NCR

| คอลัมน์ | คำอธิบาย |
|--------|-------------|
| NCR No. (เลขที่ NCR) | รหัสที่สร้างอัตโนมัติ (แบบ monospace เช่น `NCR-2026-001`) |
| Work Order (ใบสั่งงาน) | รหัส WO ที่เชื่อมโยง |
| Product (ผลิตภัณฑ์) | ชื่อผลิตภัณฑ์จาก WO ที่เชื่อมโยง |
| Defect Type (ประเภทข้อบกพร่อง) | Dimensional (มิติ), Surface (พื้นผิว), Material (วัสดุ), Functional (การทำงาน) หรือ Cosmetic (ความสวยงาม) |
| Date (วันที่) | วันที่สร้าง |
| Status (สถานะ) | ป้ายสีแสดงสถานะ: Open, In Review, Resolved, Closed |
| Disposition (การจัดการ) | Rework (ซ่อมแก้), Remake (ผลิตใหม่), Scrap (ทิ้ง), Use As-Is (ใช้ตามสภาพ), Return to Vendor (คืนผู้ขาย) (หรือ "—" หากยังไม่กำหนด) |
| Assigned To (ผู้รับผิดชอบ) | บุคคลที่รับผิดชอบการสอบสวน |

แถวสามารถคลิกได้ — คลิกแถวใดก็ได้เพื่อเปิดหน้ารายละเอียด NCR

[image: ncr-table — ภาพหน้าจอของตารางพร้อม NCR ตัวอย่าง 4-5 รายการในสถานะต่างๆ]

---

## ตัวกรองสถานะ (Status Filter)

1. คลิกดรอปดาวน์สถานะ
2. เลือกสถานะที่ต้องการกรอง (เช่น "Open" เพื่อดูเฉพาะ NCR ที่เปิดอยู่)
3. เลือก "All" เพื่อล้างตัวกรอง

> **เคล็ดลับ:** URL รองรับ Query Parameters การคลิก "Open NCRs" บน Quality Dashboard จะนำทางไปยัง `/quality/ncr?status=open` โดยมีตัวกรองถูกตั้งค่าไว้แล้ว

[image: ncr-status-filter — ภาพหน้าจอของดรอปดาวน์ตัวกรองสถานะที่ขยายออก]

---

## ป้ายสถานะ NCR (NCR Status Badges)

| สถานะ | สี |
|--------|-------|
| Open (เปิด) | เหลือง/อำพัน |
| In Review (อยู่ระหว่างตรวจสอบ) | น้ำเงิน |
| Resolved (แก้ไขแล้ว) | เขียว |
| Closed (ปิดแล้ว) | เทา |

[image: ncr-status-badges — ภาพใกล้แสดงป้ายสถานะทั้ง 4 แบบ]

---

## การค้นหา (Searching)

พิมพ์ในช่องค้นหาเพื่อกรองตาม:
- รหัส NCR (NCR code)
- รหัส WO (WO code)
- ชื่อผลิตภัณฑ์ (Product name)

[image: ncr-search — ภาพหน้าจอแสดงผลลัพธ์ที่ถูกกรอง]
