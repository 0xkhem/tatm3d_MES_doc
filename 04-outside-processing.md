# Outside Processing (งานส่งผลิตภายนอก)

## คืออะไร?

Outside Processing ครอบคลุมงานที่ต้องออกจากโรงงานไปให้ซัพภายนอกดำเนินการ เช่น การชุบผิว การอโนไดซ์ การอบชุบ การชุบโลหะ และบริการที่คล้ายกัน ระบบติดตามทุกการจัดส่งและการรับกลับเพื่อไม่ให้มีอะไรหลุดรอด

## วิธีการทำงาน

เมื่อ Work Order มีขั้นตอนที่ Process Definition ถูกตั้งค่าเป็น Outside Service ระบบจะสร้าง **Outside Processing Order (OPO)** สำหรับขั้นตอนนั้นโดยอัตโนมัติ ไม่ต้องสร้าง OPO ด้วยตนเอง — จะปรากฏขึ้นทันทีที่สร้าง WO

```
สร้าง WO ที่มีขั้นตอน Outside → OPO สร้างอัตโนมัติ (Pending)
ผู้วางแผนจัดส่งชิ้นส่วนให้ซัพ → OPO เปลี่ยนเป็น Shipped
ซัพส่งชิ้นส่วนกลับ → OPO เปลี่ยนเป็น Completed
ขั้นตอนสามารถทำเครื่องหมายว่า Complete ได้
```

หมายเหตุ: ไม่สามารถ Complete ขั้นตอน Outside ได้จนกว่า OPO จะเป็น Completed ระบบจะบล็อกไว้

## วงจรชีวิต OPO

```
Pending → Shipped → In Transit → Completed
                        │
                        └──→ Partial (รับบางส่วน ส่วนที่เหลือยังอยู่กับซัพ)
```

สถานะที่ไม่ใช่ Terminal สามารถยกเลิกได้หากขั้นตอนถูก Skip

## การแจ้งเตือนเกินกำหนด

หากชิ้นส่วนยังไม่กลับมาภายในวันที่คาดว่าจะรับกลับ OPO จะถูกแจ้งว่าเกินกำหนด:
- แถบสีแดงใน Outside Processing Panel ของ Scheduling Board
- ตัวบ่งชี้เตือนในหน้า Orders
- ปรากฏในรายการ Attention Items ของหน้าหลัก

## Packing Slip (ใบส่งของ)

ก่อนจัดส่งชิ้นส่วนให้ซัพ สามารถพิมพ์ Packing Slip ได้โดยตรงจาก OPO ประกอบด้วยที่อยู่ซัพ รายละเอียด WO และขั้นตอน ข้อมูลจำนวน และ Barcode สำหรับการติดตาม

## Vendors (ซัพ)

ซัพจัดการในแท็บ Outside Processing → Vendors แต่ละซัพมีรหัส (`VDR-0001`) ข้อมูลติดต่อ และระยะเวลานำเริ่มต้นเป็นวันทำงาน ระยะเวลานำถูกใช้โดย Scheduling Engine เพื่อสำรองช่วงเวลาโดยอัตโนมัติระหว่างขั้นตอนก่อนและหลังงานซัพ

## แนวคิดหลัก

| คำศัพท์ | ความหมาย |
|---------|----------|
| OPO | Outside Processing Order — หนึ่งรายการต่อขั้นตอน Outside ต่อ WO |
| Outside service step | ขั้นตอน WO ที่ Process Definition ถูกตั้งค่า `is_outside_service` |
| Lead time | วันทำงานที่ซัพต้องการ — ใช้บล็อกเวลาในตาราง |
| Packing Slip | เอกสารจัดส่งที่พิมพ์ได้สำหรับซัพ |
| Overdue | OPO ที่เกินวันที่คาดว่าจะรับกลับและยังไม่เสร็จสมบูรณ์ |
