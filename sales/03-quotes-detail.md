# 3. Quotes — มุมมองรายละเอียดและการดำเนินการ

## การเข้าถึงรายละเอียดใบเสนอราคา (Quote Detail)

1. จากหน้ารายการใบเสนอราคา คลิกลิงก์ **Quote No.** (เช่น `QT-2026-003`)
2. หน้ารายละเอียดใบเสนอราคาจะโหลดที่ `/sales/quotes/:id`

[image: quote-detail-full — ภาพหน้าจอเต็มของหน้า Quote Detail แสดงส่วนหัวพร้อมปุ่มดำเนินการ การ์ดข้อมูล และการ์ดรายการสินค้าด้านล่าง]

---

## เค้าโครงหน้า (Page Layout)

### แถบส่วนหัว (Header Bar)

ส่วนหัวแสดง:

- **ลูกศรย้อนกลับ** (←) — กลับไปยังรายการใบเสนอราคา
- **รหัสใบเสนอราคา** (เช่น `QT-2026-003`) ในฟอนต์ monospace
- **ชื่อลูกค้า** ด้านล่างรหัส
- **ป้ายสถานะ** (Status badge) — แสดงสีตามสถานะ (Draft, Sent, Accepted ฯลฯ)
- **ปุ่มดำเนินการ** (ด้านขวา) — เปลี่ยนตามสถานะปัจจุบัน

[image: quote-detail-header — ภาพระยะใกล้ของแถบส่วนหัวแสดงรหัสใบเสนอราคา ชื่อลูกค้า ป้ายสถานะ และปุ่มดำเนินการ]

## ปุ่มดำเนินการตามสถานะ (Action Buttons by Status)

การดำเนินการที่มีจะเปลี่ยนไปตามสถานะปัจจุบันของใบเสนอราคา:

### เมื่อสถานะ = Draft (ร่าง)

| ปุ่ม | ไอคอน | การดำเนินการ |
|------|--------|-------------|
| Print/PDF (พิมพ์/PDF) | 📄 | เปิดหน้าดู PDF |
| Share Link (แชร์ลิงก์) | 🔗 | สร้าง URL สาธารณะและคัดลอกไปยังคลิปบอร์ด |
| Duplicate (คัดลอก) | 📋 | สร้างสำเนาของใบเสนอราคานี้พร้อมรหัสใหม่ |
| Edit (แก้ไข) | ✏ | เปิดแบบฟอร์มแก้ไข |
| Send to Customer (ส่งให้ลูกค้า) | ✉ | เปลี่ยนสถานะจาก Draft → Sent |
| Cancel Quote (ยกเลิกใบเสนอราคา) | — | เปลี่ยนสถานะจาก Draft → Cancelled |

[image: quote-actions-draft — ภาพระยะใกล้ของปุ่มดำเนินการเมื่อใบเสนอราคาอยู่ในสถานะ Draft]

### เมื่อสถานะ = Sent (ส่งแล้ว)

| ปุ่ม | ไอคอน | การดำเนินการ |
|------|--------|-------------|
| Print/PDF (พิมพ์/PDF) | 📄 | เปิดหน้าดู PDF |
| Share Link (แชร์ลิงก์) | 🔗 | สร้าง URL สาธารณะ |
| Duplicate (คัดลอก) | 📋 | สร้างสำเนา |
| Mark Accepted (ทำเครื่องหมายว่ายอมรับ) | ✅ | เปลี่ยนสถานะ Sent → Accepted (ปุ่มสีเขียว) |
| Mark Rejected (ทำเครื่องหมายว่าปฏิเสธ) | ✖ | เปลี่ยนสถานะ Sent → Rejected |
| Mark Expired (ทำเครื่องหมายว่าหมดอายุ) | ⏰ | เปลี่ยนสถานะ Sent → Expired |
| Cancel (ยกเลิก) | — | เปลี่ยนสถานะ Sent → Cancelled |

[image: quote-actions-sent — ภาพระยะใกล้ของปุ่มดำเนินการเมื่อใบเสนอราคาอยู่ในสถานะ Sent เน้นปุ่ม "Mark Accepted" สีเขียว]

### เมื่อสถานะ = Accepted (ยอมรับ) — ยังไม่แปลง

| Share Link (แชร์ลิงก์) | 🔗 | สร้าง URL สาธารณะ |
| Duplicate (คัดลอก) | 📋 | สร้างสำเนา |
| Convert to Order (แปลงเป็นใบสั่งขาย) | 🛒 | สร้าง Sales Order จากใบเสนอราคานี้ |

[image: quote-actions-accepted — ภาพระยะใกล้แสดงปุ่ม "Convert to Order" สำหรับใบเสนอราคาที่ Accepted]

### เมื่อสถานะ = Accepted (ยอมรับ) — แปลงแล้ว

|------|--------|-------------|

| Share Link (แชร์ลิงก์) | 🔗 | สร้าง URL สาธารณะ |

## การ์ดข้อมูลใบเสนอราคา (Quote Information Card)

ด้านล่างส่วนหัว การ์ดจะแสดงรายละเอียดใบเสนอราคาในรูปแบบตาราง:

| ฟิลด์ | คำอธิบาย |
|-------|---------|
| Customer (ลูกค้า) | ชื่อลูกค้า |
| Created Date (วันที่สร้าง) | วันที่สร้างใบเสนอราคา |
| Valid Until (ใช้ได้ถึง) | วันหมดอายุ |
| Lead Time (ระยะเวลาการผลิต) | จำนวนวันทำการ |
| Shipping Method (วิธีการจัดส่ง) | เช่น "DHL" |
| Shipping Cost (ค่าจัดส่ง) | จำนวนเงินค่าจัดส่ง |
| Payment Term (เงื่อนไขการชำระเงิน) | เช่น "Full payment in advance" (ชำระเงินล่วงหน้าเต็มจำนวน) |
| Trade Term (เงื่อนไขการค้า) | เช่น "EXW (Ex Works)" |
| Notes (หมายเหตุ) | หมายเหตุเพิ่มเติม |
| Delivery Address (ที่อยู่จัดส่ง) | ที่อยู่จัดส่งของลูกค้า (จากข้อมูลลูกค้า) |
| Invoice Address (ที่อยู่ออกใบแจ้งหนี้) | ที่อยู่ออกใบแจ้งหนี้ของลูกค้า (จากข้อมูลลูกค้า) |
| Converted Sales Order (ใบสั่งขายที่แปลงแล้ว) | ลิงก์ไปยัง SO หากใบเสนอราคาได้ถูกแปลงแล้ว |

[image: quote-info-card — ภาพหน้าจอของการ์ด Quote Details แสดงฟิลด์ทั้งหมดในรูปแบบตาราง 4 คอลัมน์]

## การ์ดรายการสินค้า (Line Item Cards)

แต่ละรายการสินค้าแสดงเป็นการ์ดสองคอลัมน์:

### คอลัมน์ซ้าย — รายละเอียดรายการ

- **ภาพขนาดย่อแบบเขียนแบบ** (คลิกเพื่อเปิดดูภาพขนาดเต็ม)
- Part/Product Name (ชื่อชิ้นส่วน/ผลิตภัณฑ์)
- Material Spec (ข้อกำหนดวัสดุ)
- Surface Finish (การตกแต่งผิว)
- Quantity (จำนวน), Unit (หน่วย), Unit Price (ราคาต่อหน่วย), Lead Time (ระยะเวลาการผลิต), Line Total (ยอดรวมต่อรายการ)
- Notes (หมายเหตุ) — ถ้ามี
- **Delivery Promise Signal** (ตัวบ่งชี้ความเป็นไปได้ของการจัดส่ง — เขียว/เหลือง/แดง)

### คอลัมน์ขวา — กระบวนการผลิต (Manufacturing Processes)

- รายการขั้นตอนกระบวนการตามลำดับ
- แต่ละขั้นตอนแสดง: หมายเลขลำดับ ชื่อกระบวนการ (พร้อมรหัส) ต้นทุนต่อหน่วย
- ต้นทุนกระบวนการรวมแสดงที่ด้านบน

[image: quote-line-item-card — ภาพหน้าจอของการ์ดรายการสินค้าเดียวแสดงรูปแบบสองคอลัมน์ พร้อมภาพขนาดย่อแบบเขียนแบบ รายละเอียดทางซ้าย และขั้นตอนกระบวนการทางขวา]

## ยอดรวมใบเสนอราคา (Quote Totals)

ที่ด้านล่างของส่วนรายการสินค้า:

- **Subtotal** (ยอดย่อย) — ผลรวมของยอดรวมรายการสินค้าทั้งหมด
- **Shipping** (ค่าจัดส่ง) — ค่าจัดส่ง (หากมากกว่า 0)
- **VAT 7%** (ภาษีมูลค่าเพิ่ม 7%) — จำนวนภาษี (หากรวมภาษี)
- **Total** (ยอดรวมทั้งหมด) — ยอดรวมสุทธิ

[image: quote-detail-totals — ภาพระยะใกล้ของส่วนยอดรวมที่ด้านล่างของรายการสินค้า]

## ขั้นตอนการทำงานหลักจากหน้ารายละเอียด (Key Workflows)

### การส่งใบเสนอราคาให้ลูกค้า

1. เปิดใบเสนอราคา **Draft**
2. คลิกปุ่ม **Send to Customer**
3. สถานะจะเปลี่ยนเป็น **Sent**
4. การแจ้งเตือนยืนยัน: "Quote updated to Sent." (อัปเดตใบเสนอราคาเป็น Sent แล้ว)

> **สิ่งที่เกิดขึ้น:** ใบเสนอราคาจะถูกล็อคไม่ให้แก้ไข ลูกค้าสามารถดูผ่านลิงก์แชร์ได้

### การทำเครื่องหมายว่ายอมรับ

1. เปิดใบเสนอราคา **Sent**
2. คลิกปุ่ม **Mark Accepted** สีเขียว
3. สถานะจะเปลี่ยนเป็น **Accepted**
4. ปุ่ม **Convert to Order** จะปรากฏ

### การแปลงใบเสนอราคาเป็นใบสั่งขาย

1. เปิดใบเสนอราคา **Accepted** (ที่ยังไม่ได้แปลง)
2. คลิก **Convert to Order**
3. ระบบจะดำเนินการโดยอัตโนมัติ:
   - สร้าง Sales Order ใหม่พร้อมรหัสที่สร้างขึ้น (เช่น `SO-2026-003`)
   - คัดลอกรายการสินค้าทั้งหมดจากใบเสนอราคาไปยัง SO
   - เชื่อมโยง SO กลับไปยังใบเสนอราคานี้ผ่าน `source_quote_id`
   - เชื่อมโยงรายการสินค้า SO แต่ละรายการไปยังรายการใบเสนอราคาต้นทางผ่าน `source_quote_item_id`
4. การแจ้งเตือนยืนยัน: "Sales Order SO-2026-003 created." (สร้างใบสั่งขาย SO-2026-003 แล้ว)
5. ปุ่มจะเปลี่ยนเป็น **View Sales Order**

[image: quote-convert-success — ภาพหน้าจอแสดงการแจ้งเตือนความสำเร็จหลังแปลงใบเสนอราคาเป็นใบสั่งขาย]

### การคัดลอกใบเสนอราคา (Duplicate)

1. คลิก **Duplicate** บนใบเสนอราคาใด ๆ (ทุกสถานะ)
2. ใบเสนอราคาใหม่จะถูกสร้างเป็นสำเนาในสถานะ **Draft** พร้อมรหัสใหม่
3. คุณจะถูกนำไปยังหน้ารายละเอียดของใบเสนอราคาใหม่
4. แก้ไขสำเนาตามต้องการ

> **กรณีการใช้งาน:** สร้างใบเสนอราคาฉบับปรับปรุงสำหรับลูกค้ารายเดิมได้อย่างรวดเร็ว หรือนำเทมเพลตใบเสนอราคาไปใช้กับลูกค้ารายอื่น
