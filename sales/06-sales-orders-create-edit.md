# 6. คำสั่งซื้อ — สร้างและแก้ไข

## สองวิธีในการสร้างคำสั่งซื้อ

### เส้นทาง A: แปลงจากใบเสนอราคาที่ได้รับการยอมรับ (แนะนำ)

1. เปิดใบเสนอราคาที่มีสถานะ **Accepted** ในหน้ารายละเอียด Quote
2. คลิก **Convert to Order**
3. Sales Order จะถูกสร้างอัตโนมัติพร้อมรายการสินค้า, กระบวนการ และราคาทั้งหมดที่คัดลอกจากใบเสนอราคา
4. SO จะเชื่อมโยงกับใบเสนอราคาต้นทางเพื่อการติดตามย้อนกลับ

> นี่คือเส้นทางที่พบบ่อยที่สุด ดู [Quotes — Detail View & Actions](./03-quotes-detail.md) สำหรับรายละเอียด

### เส้นทาง B: สร้างโดยตรง

1. ไปที่ **Sales → Sales Order**
2. คลิก **+ Sales Order**
3. ฟอร์มสร้างจะเปิดที่ `/sales-order/create`

[image: sales-order-create-page — ภาพหน้าจอเต็มของฟอร์ม Create Sales Order ที่ว่างเปล่า]

## ฟอร์มคำสั่งซื้อ — ฟิลด์ส่วนหัว

| ฟิลด์ | จำเป็น | คำอธิบาย |
|-------|----------|-------------|
| Order Code | ใช่ | สร้างอัตโนมัติ (เช่น `SO-2026-005`) สามารถแก้ไขด้วยตนเองได้ |
| Customer | ใช่ | Dropdown เพื่อเลือกจากลูกค้าที่มีอยู่ |
| Customer PO Reference | ไม่ | หมายเลขใบสั่งซื้อของลูกค้าสำหรับการอ้างอิงข้าม |
| Order Date | ใช่ | ค่าเริ่มต้นเป็นวันนี้ |
| Delivery Date | ใช่ | วันส่งมอบเป้าหมาย ต้องเป็นวันที่เดียวกับหรือหลัง Order Date |
| Status | ใช่ | ค่าเริ่มต้นเป็น "Draft" สำหรับคำสั่งซื้อใหม่ |

[image: sales-order-form-header — ภาพใกล้ของฟิลด์ส่วนหัวแสดง Dropdown ลูกค้า, อ้างอิง PO, วันที่ และสถานะ]

## การเพิ่มรายการสินค้า

ส่วนรายการสินค้าทำงานคล้ายกับฟอร์ม Quote:

### ลากและวางแบบ

1. ลากไฟล์แบบวิศวกรรม (PDF, TIFF, PNG, STEP) ลงบนพื้นที่วาง
2. แต่ละไฟล์จะสร้างรายการสินค้าใหม่พร้อมหมายเลขแบบที่ดึงจากชื่อไฟล์
3. Thumbnail และมุมมอง Isometric จะถูกสร้างอัตโนมัติ

[image: sales-order-drawing-drop — ภาพหน้าจอของพื้นที่วางและรายการสินค้าที่สร้างอัตโนมัติหลังจากวางไฟล์]

### การป้อนข้อมูลด้วยตนเอง

1. คลิก **+ Add Item** เพื่อเพิ่มรายการสินค้าว่างเปล่า
2. กรอกฟิลด์ด้วยตนเอง

---

## ฟิลด์รายการสินค้า

แต่ละรายการสินค้ามีฟิลด์เหล่านี้:

| ฟิลด์ | จำเป็น | คำอธิบาย |
|-------|----------|-------------|
| Part/Product Name | ใช่ | ชื่อชิ้นส่วน |
| Product Code | ไม่ | หมายเลขชิ้นส่วนเพิ่มเติม |
| Drawing Number | ไม่ | กรอกอัตโนมัติจากชื่อไฟล์หรือป้อนด้วยตนเอง |
| Specification | ไม่ | ข้อกำหนดทางเทคนิค |
| Material Spec | ไม่ | วัสดุ (เช่น "Al 7075-T6") |
| Surface Finish | ไม่ | การบำบัดผิว |
| Quantity | ใช่ | จำนวนเต็มบวก |
| Unit | ใช่ | ค่าเริ่มต้นเป็น "pcs" |
| Unit Price | ใช่ | ราคาต่อหน่วย (≥ 0) |
| Lead Time (days) | ไม่ | ระยะเวลานำต่อรายการในวันทำงาน |
| Notes | ไม่ | หมายเหตุเฉพาะรายการ |

[image: sales-order-line-item — ภาพใกล้ของการ์ดรายการสินค้าเดียวพร้อมฟิลด์ที่กรอกแล้ว]

## กระบวนการผลิตต่อรายการ

แต่ละรายการสินค้า SO สามารถมีขั้นตอนกระบวนการผลิต เช่นเดียวกับใบเสนอราคา:

1. คลิก **+ Add Process** ในส่วนกระบวนการของรายการ
2. เลือก **Process Definition** จาก Dropdown
3. ตั้งค่า **Sequence**, **Cost per Unit** และ **Notes** เพิ่มเติม
4. ลากเพื่อเรียงลำดับใหม่, คลิกถังขยะเพื่อลบ

คุณยังสามารถ **Load from Route** เพื่อเติมข้อมูลจาก Process Route ที่กำหนดไว้ล่วงหน้า

[image: sales-order-item-processes — ภาพใกล้ของส่วนขั้นตอนกระบวนการบนรายการสินค้า]

> **เมื่อแปลงจากใบเสนอราคา:** กระบวนการทั้งหมดจะถูกคัดลอกอัตโนมัติจากรายการใบเสนอราคา ไม่ต้องป้อนข้อมูลด้วยตนเอง

## สัญญาณ Delivery Promise

ฟอร์มแสดงสัญญาณความเป็นไปได้ในการส่งมอบ:

- **สัญญาณต่อรายการ** — ถัดจากแต่ละรายการสินค้าที่มีกระบวนการและระยะเวลานำ
- **สัญญาณรวม** — ที่ด้านบนของฟอร์ม สรุปความเป็นไปได้ในทุกรายการ

สัญญาณเหล่านี้ตรวจสอบกำลังการผลิตของโรงงานเทียบกับวันส่งมอบที่ร้องขอและแสดงตัวบ่งชี้เขียว/เหลือง/แดง

[image: sales-order-delivery-signals — ภาพหน้าจอแสดงทั้งสัญญาณรวมด้านบนและสัญญาณต่อรายการบนรายการสินค้า]

---

## การบันทึกคำสั่งซื้อ

1. ตรวจสอบฟิลด์และรายการสินค้าทั้งหมด
2. คลิก **Save** (หรือ **Create Sales Order** สำหรับคำสั่งซื้อใหม่)
3. คำสั่งซื้อถูกบันทึกในสถานะที่เลือก (โดยทั่วไปคือ **Draft**)
4. Toast สำเร็จจะปรากฏ

---

## การแก้ไขคำสั่งซื้อที่มีอยู่

1. จากรายการ Sales Order คลิกไอคอน **ดินสอ** (✏) บนแถวคำสั่งซื้อใดก็ได้
2. ฟอร์มแก้ไขโหลดที่ `/sales-order/edit/:id` พร้อมข้อมูลที่กรอกไว้ล่วงหน้า
3. ทำการเปลี่ยนแปลงและคลิก **Save**

[image: sales-order-edit-form — ภาพหน้าจอของฟอร์มแก้ไขพร้อมข้อมูลที่กรอกไว้ล่วงหน้า]

---

## ขั้นตอนทั่วไปหลังจากสร้างคำสั่งซื้อ

1. **สร้าง** SO (จากการแปลงใบเสนอราคาหรือการป้อนโดยตรง) → สถานะ: **Draft**
2. **ตรวจสอบ** รายละเอียดคำสั่งซื้อและรายการสินค้า
3. **ยืนยัน** คำสั่งซื้อ → เปลี่ยนสถานะเป็น **Confirmed** ผ่าน Dropdown สถานะในหน้ารายการ
4. **สร้าง Work Orders** → คลิกปุ่ม "+ Work Orders" (ดูส่วนถัดไป)
5. สถานะเปลี่ยนเป็น **In Production** อัตโนมัติ
