# 5. คำสั่งซื้อ — รายการและการจัดการ

## การเข้าถึงรายการคำสั่งซื้อ

1. คลิก **Sales** ในแถบด้านข้างซ้าย
2. คลิก **Sales Order**
3. หน้า Sales Order โหลดที่ `/sales-order`

[image: sales-order-list-page — ภาพหน้าจอเต็มของหน้ารายการ Sales Order แสดงแถบเครื่องมือ, ตัวกรองสถานะ และตารางพร้อมตัวอย่างคำสั่งซื้อ]

---

## เค้าโครงหน้า (Page Layout)

### แถบเครื่องมือ (Toolbar)

- ปุ่ม **+ Sales Order** (ซ้าย) — สร้างคำสั่งซื้อใหม่โดยตรง
- ข้อความอธิบาย: "Manage customer sales orders"
- **Dropdown ตัวกรองสถานะ** — กรองตาม: All Statuses, Draft, Confirmed, In Production, Completed, Cancelled
- **ช่องค้นหา** (ขวา) — กรองตามรหัสคำสั่งซื้อหรือชื่อลูกค้า

[image: sales-order-toolbar — ภาพใกล้ของแถบเครื่องมือแสดงปุ่ม "+ Sales Order", Dropdown ตัวกรองสถานะ และช่องค้นหา]

---

### ตารางคำสั่งซื้อ

ตารางแสดงคำสั่งซื้อทั้งหมดพร้อมคอลัมน์เหล่านี้:

| คอลัมน์ | คำอธิบาย |
|--------|-------------|
| ▶ (ขยาย) | คลิกเพื่อขยายและแสดงรายการสินค้าแบบ Inline |
| Order No. | รหัสที่สร้างอัตโนมัติ (เช่น `SO-2026-001`) แบบ Monospace |
| Customer | ชื่อลูกค้า |
| Order Date | วันที่สั่งซื้อ |
| Delivery Date | วันส่งมอบเป้าหมายพร้อมตัวบ่งชี้เกินกำหนด/เตือน |
| Items | จำนวนรายการสินค้า |
| Total Amount | ยอดรวมที่คำนวณ |
| Production Progress | แถบความคืบหน้าแสดง % ความสำเร็จในทุก Work Order ที่เชื่อมโยง |
| Status | Dropdown สำหรับการเปลี่ยนสถานะ (หรือป้ายสถานะคงที่สำหรับสถานะสุดท้าย) |
| Operate | ปุ่มการดำเนินการ: + Work Orders, Edit, Delete |

[image: sales-order-table — ภาพหน้าจอของตารางพร้อมตัวอย่าง 3–4 แถวแสดงสถานะต่างกันและเค้าโครงคอลัมน์]

---

## การกรองตามสถานะ

1. คลิก **Dropdown ตัวกรองสถานะ** ในแถบเครื่องมือ
2. เลือกสถานะ (เช่น "Confirmed") เพื่อแสดงเฉพาะคำสั่งซื้อในสถานะนั้น
3. เลือก "All Statuses" เพื่อล้างตัวกรอง

[image: sales-order-status-filter — ภาพหน้าจอของ Dropdown ตัวกรองสถานะที่ขยายออกแสดงตัวเลือกสถานะทั้งหมด]

---

## การค้นหาคำสั่งซื้อ

1. คลิกช่องค้นหา
2. พิมพ์รหัสคำสั่งซื้อ (เช่น `SO-2026`) หรือชื่อลูกค้า (เช่น `Ford`)
3. ตารางกรองแบบเรียลไทม์

---

## ตัวบ่งชี้วันส่งมอบ

คอลัมน์ Delivery Date ใช้ตัวบ่งชี้สีที่มีรหัส:

| ตัวบ่งชี้ | สี | ความหมาย |
|-----------|-------|---------|
| ⚠ แดง + "Xd overdue" | แดง | วันส่งมอบผ่านไปแล้ว |
| ⚠ อำพัน + "Xd left" | อำพัน | วันส่งมอบอยู่ภายใน 7 วัน |
| วันที่ + "(Xd)" | ปกติ | มากกว่า 7 วันจนถึงการส่งมอบ |

[image: sales-order-delivery-dates — ภาพใกล้ของคอลัมน์ Delivery Date แสดงตัวอย่างวันที่เกินกำหนด (แดง), เตือน (อำพัน) และปกติ]

---

## ความคืบหน้าการผลิต

คอลัมน์ Production Progress แสดงความสำเร็จโดยรวมของ Work Order ที่เชื่อมโยง:

- **"No production started"** — ยังไม่มีการสร้าง Work Order
- **แถบความคืบหน้า + เปอร์เซ็นต์** — แสดง % ของขั้นตอน WO ที่เสร็จสิ้น
- **ป้าย "Delivered"** (เขียว) — Work Order ทั้งหมดเสร็จสมบูรณ์

[image: sales-order-progress — ภาพใกล้ของคอลัมน์ Production Progress แสดงสถานะต่างกัน: ไม่มีการผลิต, แถบความคืบหน้า 45% และป้าย Delivered]

---

## การเปลี่ยนสถานะคำสั่งซื้อ

สำหรับคำสั่งซื้อที่มีการเปลี่ยนสถานะที่ใช้ได้ คอลัมน์ Status จะแสดง **Dropdown** แทนป้ายสถานะคงที่:

1. คลิก Dropdown สถานะบนแถวคำสั่งซื้อ
2. สถานะปัจจุบันจะแสดง (ปิดใช้งาน)
3. การเปลี่ยนสถานะที่ใช้ได้จะแสดงพร้อมคำนำหน้าลูกศร (เช่น "→ Confirmed")
4. เลือกสถานะใหม่
5. Toast ยืนยันการเปลี่ยนแปลง

### กฎการเปลี่ยนสถานะ

| สถานะปัจจุบัน | สามารถเปลี่ยนไปยัง |
|----------------|-------------------|
| Draft | Confirmed, Cancelled |
| Confirmed | In Production, Cancelled |
| In Production | Completed, Cancelled |
| Completed | (ไม่มีการเปลี่ยนสถานะ — สถานะสุดท้าย) |
| Cancelled | (ไม่มีการเปลี่ยนสถานะ — สถานะสุดท้าย) |

[image: sales-order-status-dropdown — ภาพใกล้ของ Dropdown สถานะที่ขยายออกบนคำสั่งซื้อ "Draft" แสดงตัวเลือก "→ Confirmed" และ "→ Cancelled"]

> **หมายเหตุ:** สถานะ "In Production" โดยทั่วไปจะถูกตั้งค่าอัตโนมัติเมื่อสร้าง Work Order คุณยังสามารถเปลี่ยนสถานะด้วยตนเองได้

---

## การขยายรายการสินค้า

1. คลิกปุ่ม **▶ ลูกศร** ทางซ้ายของแถวคำสั่งซื้อใดก็ได้
2. แถวจะขยายเพื่อแสดงตารางรายการสินค้าแบบซ้อน
3. คลิกอีกครั้งเพื่อยุบ

### ตารางรายการสินค้าที่ขยาย

| คอลัมน์ | คำอธิบาย |
|--------|-------------|
| Thumbnail | มุมมอง Isometric หรือ Thumbnail ของแบบ (สี่เหลี่ยมเล็ก) |
| Product Name | ชื่อชิ้นส่วน/ผลิตภัณฑ์ |
| Qty | จำนวนและหน่วย |
| Unit Price | ราคาต่อหน่วย |
| Total | ยอดรวมต่อรายการ |
| Processes | การแสดงผลกระบวนการเชิงภาพแสดงวงกลมหมายเลขที่เชื่อมต่อด้วยเส้น พร้อมชื่อกระบวนการด้านล่างแต่ละขั้นตอน |
| Status | สถานะรายการสินค้า |

[image: sales-order-expanded-items — ภาพหน้าจอของแถวคำสั่งซื้อที่ขยายแสดงตารางรายการสินค้าแบบซ้อนพร้อม Thumbnail และการแสดงผลกระบวนการ]

---

## การแก้ไขคำสั่งซื้อ

1. คลิกไอคอน **ดินสอ** (✏) ในคอลัมน์ Operate
2. ฟอร์มแก้ไขจะเปิดที่ `/sales-order/edit/:id`
3. ทำการเปลี่ยนแปลงและบันทึก

---

## การลบคำสั่งซื้อ

เฉพาะคำสั่งซื้อในสถานะ **Draft** หรือ **Confirmed** เท่านั้นที่สามารถลบได้

1. คลิกไอคอน **ถังขยะ** (🗑) ในคอลัมน์ Operate
2. กล่องโต้ตอบยืนยันจะปรากฏพร้อมรหัสคำสั่งซื้อและชื่อลูกค้า
3. คลิก **Delete** เพื่อยืนยัน
4. Toast สำเร็จจะปรากฏ

[image: sales-order-delete-dialog — ภาพหน้าจอของกล่องโต้ตอบยืนยันการลบ]

> **สำคัญ:** ถ้าคำสั่งซื้อถูกอ้างอิงโดยระเบียนโปรเจกต์ (เช่น มีการสร้าง Work Order จากคำสั่งซื้อนั้นแล้ว) การลบจะล้มเหลวพร้อมข้อความแสดงข้อผิดพลาด: "This sales order is referenced by project records and cannot be deleted."

[image: sales-order-delete-error — ภาพหน้าจอของ Toast ข้อผิดพลาดเมื่อพยายามลบคำสั่งซื้อที่ถูกอ้างอิง]
