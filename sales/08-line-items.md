# 8. รายการสินค้า — มุมมองรวม

## การเข้าถึงหน้า Line Items

1. คลิก **Sales** ในแถบด้านข้างซ้าย
2. คลิก **Line Items**
3. หน้าโหลดที่ `/sales/line-items`

[image: line-items-page — ภาพหน้าจอเต็มของหน้า Line Items แสดงแถบค้นหาและตารางพร้อมตัวอย่างรายการสินค้าจากหลายคำสั่งซื้อ]

---

## วัตถุประสงค์

หน้า Line Items ให้ **มุมมองข้ามคำสั่งซื้อ** ของทุกรายการสินค้าในทุกคำสั่งซื้อ มีประโยชน์เมื่อคุณต้องการ:

- ค้นหาชิ้นส่วนเฉพาะในทุกคำสั่งซื้อ
- ตรวจสอบวันส่งมอบสำหรับรายการจากลูกค้าต่างกัน
- ดูภาพรวมรวดเร็วของรายการทั้งหมดในไปป์ไลน์
- ค้นหาข้อกำหนดวัสดุหรือราคาโดยไม่ต้องเปิดคำสั่งซื้อแต่ละรายการ

---

## เค้าโครงหน้า (Page Layout)

### แถบค้นหา

- อยู่ที่ด้านบนของหน้า
- กรองตาม: ชื่อผลิตภัณฑ์, รหัสคำสั่งซื้อ SO, ชื่อลูกค้า หรือข้อกำหนดวัสดุ

[image: line-items-search — ภาพใกล้ของแถบค้นหาพร้อมคำค้นหาที่ป้อน]

### ตารางรายการสินค้า

| คอลัมน์ | คำอธิบาย |
|--------|-------------|
| Thumbnail | มุมมอง Isometric หรือ Thumbnail ของแบบขนาดเล็ก |
| Product | ชื่อชิ้นส่วน/ผลิตภัณฑ์ |
| SO Code | รหัสคำสั่งซื้อหลัก (เช่น `SO-2026-001`) |
| Customer | ชื่อลูกค้าจาก Sales Order ที่เชื่อมโยง |
| Qty | จำนวนและหน่วย |
| Material | ข้อกำหนดวัสดุ |
| Surface | การบำบัดผิว |
| Unit Price | ราคาต่อหน่วย |
| Delivery Date | จาก Sales Order หลัก พร้อมตัวบ่งชี้เกินกำหนด/เตือน |
| SO Status | ป้ายสถานะของ Sales Order หลัก |

[image: line-items-table — ภาพหน้าจอของตารางแสดงรายการสินค้า 4–5 รายการจาก Sales Order ต่างกันพร้อมตัวบ่งชี้วันส่งมอบต่างๆ]

---

## ตัวบ่งชี้วันส่งมอบ

คอลัมน์ Delivery Date ใช้ระบบสีเดียวกับหน้า Sales Order:

- 🔴 **แดง** พร้อม ⚠ — เกินกำหนด (เช่น "15 Mar 2026 (3d overdue)")
- 🟡 **อำพัน** พร้อม ⚠ — ครบกำหนดภายใน 7 วัน (เช่น "10 Apr 2026 (5d left)")
- ข้อความปกติ — มากกว่า 7 วัน (เช่น "25 Apr 2026 (22d)")

[image: line-items-delivery-indicators — ภาพใกล้ของคอลัมน์ Delivery Date แสดงตัวอย่างวันที่แดง, อำพัน และปกติ]

---

## การดูรายละเอียดรายการ

1. คลิกแถวใดก็ได้ในตาราง
2. **แผ่นรายละเอียด** จะเลื่อนเข้ามาจากด้านขวา
3. แผ่นแสดงรายละเอียดรายการทั้งหมดในรูปแบบที่มีโครงสร้าง:

| รายละเอียด | คำอธิบาย |
|--------|-------------|
| Product Name | ชื่อชิ้นส่วน |
| SO Code | รหัส Sales Order หลัก |
| Customer | ชื่อลูกค้า |
| Quantity | จำนวนและหน่วย |
| Unit Price | ราคาต่อหน่วย |
| Material Spec | ข้อกำหนดวัสดุ |
| Surface Finish | การบำบัดผิว |
| Specification | ข้อกำหนดทางเทคนิค |
| Lead Time | ระยะเวลานำในวัน |
| Drawing | มุมมอง Isometric หรือ Thumbnail (ถ้ามี) |

4. คลิกนอกแผ่นหรือกด Escape เพื่อปิด

[image: line-items-detail-sheet — ภาพหน้าจอของแผ่นรายละเอียดที่เลื่อนเข้ามาจากด้านขวา แสดงรายละเอียดรายการทั้งหมด]

## การค้นหาและกรอง

การค้นหาทำงานในหลายฟิลด์พร้อมกัน:

1. พิมพ์ `Al 6061` → แสดงรายการทั้งหมดที่มี "Al 6061" ในข้อกำหนดวัสดุ
2. พิมพ์ `SO-2026-003` → แสดงรายการทั้งหมดจาก Sales Order นั้น
3. พิมพ์ `XPeng` → แสดงรายการทั้งหมดสำหรับ XPeng Motors
4. พิมพ์ `Motor Housing` → แสดงรายการทั้งหมดที่มีชื่อผลิตภัณฑ์นั้น

[image: line-items-search-results — ภาพหน้าจอแสดงผลลัพธ์ที่กรองหลังจากค้นหาวัสดุหรือลูกค้าเฉพาะ]
