# ทีมขาย (Sales Team) — คู่มือการใช้งาน

## ภาพรวม

คู่มือนี้ครอบคลุมทุกงานที่ทีมขาย (Sales Team) ดำเนินการในระบบ TAM3D MES ตั้งแต่การรับคำสอบถามจากลูกค้าไปจนถึงการส่งมอบคำสั่งซื้อที่ยืนยันแล้วให้ฝ่ายผลิต

### การนำทาง (Navigation)

ในแถบด้านข้างซ้าย (left sidebar) ให้ขยายเมนู **Sales** เพื่อดูหน้าย่อยสามหน้า:

| รายการเมนู | เส้นทาง (Path) | วัตถุประสงค์ |
|------------|----------------|-------------|
| Quotes (ใบเสนอราคา) | `/sales/quotes` | สร้าง ส่ง และจัดการใบเสนอราคา/RFQ |
| Sales Order (ใบสั่งขาย) | `/sales-order` | จัดการคำสั่งซื้อของลูกค้าที่ยืนยันแล้ว |
| Line Items (รายการสินค้า) | `/sales/line-items` | มุมมองรวมของรายการสินค้าทั้งหมดในทุกใบสั่งขาย |

[image: sidebar-sales-menu — ภาพหน้าจอของแถบด้านข้างซ้ายที่แสดงส่วน "Sales" ขยายออก พร้อมรายการเมนูย่อย Quotes, Sales Order และ Line Items]

### ขั้นตอนการขายแบบครบวงจร (End-to-End Sales Flow)

ขั้นตอนการขายมาตรฐานเป็นไปตามลำดับนี้:

```
Customer Inquiry (คำสอบถามจากลูกค้า)
      ↓
  Create Quote (สร้างใบเสนอราคา - Draft)
      ↓
  Send to Customer (ส่งให้ลูกค้า - Sent)
      ↓
  Customer Accepts / Rejects (ลูกค้ายอมรับ / ปฏิเสธ)
      ↓
  Convert Accepted Quote → Sales Order (แปลงใบเสนอราคาที่ยอมรับ → ใบสั่งขาย)
      ↓
  Confirm Sales Order (ยืนยันใบสั่งขาย)
      ↓
  Create Work Orders → Production (สร้างใบสั่งผลิต → เข้าสู่การผลิต)
```

[image: sales-flow-diagram — แผนภาพหรือภาพหน้าจอแสดงขั้นตอนจาก Quote → Sales Order → Work Orders พร้อมป้ายสถานะในแต่ละขั้นตอน]

### อ้างอิงสถานะ (Status Reference)

**สถานะของ Quote (ใบเสนอราคา):**

| สถานะ | สี | ความหมาย |
|--------|-----|---------|
| Draft (ร่าง) | เทา | กำลังจัดเตรียม แก้ไขได้ |
| Sent (ส่งแล้ว) | น้ำเงิน | ส่งให้ลูกค้าแล้ว รอการตอบกลับ |
| Accepted (ยอมรับ) | เขียว | ลูกค้ายอมรับใบเสนอราคา |
| Rejected (ปฏิเสธ) | แดง | ลูกค้าปฏิเสธใบเสนอราคา |
| Expired (หมดอายุ) | เหลือง | เลยระยะเวลาที่กำหนดแล้ว |
| Cancelled (ยกเลิก) | เทา | ยกเลิกด้วยตนเอง |

**สถานะของ Sales Order (ใบสั่งขาย):**

| สถานะ | สี | ความหมาย |
|--------|-----|---------|
| Draft (ร่าง) | เทา | กำลังจัดเตรียม แก้ไขได้ |
| Confirmed (ยืนยันแล้ว) | น้ำเงิน | อนุมัติแล้ว พร้อมสำหรับการผลิต |
| In Production (อยู่ระหว่างผลิต) | เหลือง | สร้างใบสั่งผลิตแล้ว |
| Completed (เสร็จสิ้น) | เขียว | ใบสั่งผลิตทั้งหมดเสร็จเรียบร้อย |
| Cancelled (ยกเลิก) | แดง | คำสั่งซื้อถูกยกเลิก |
