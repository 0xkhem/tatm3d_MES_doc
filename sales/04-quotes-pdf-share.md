# 4. Quotes — PDF, พิมพ์ และแชร์

1. จากหน้า Quote Detail คลิก **Print/PDF** (ไอคอน 📄)
2. หน้า PDF Viewer จะเปิดที่ `/sales/quotes/:id/pdf`

### ตัวควบคุมการซูม (Zoom Controls)

- **ช่วงการซูม:** 50% ถึง 200%
- **การซูมเริ่มต้น:** 85%
- คลิกข้อความ **เปอร์เซ็นต์การซูม** เพื่อรีเซ็ตเป็นค่าเริ่มต้น
- ใช้ **Fit to Width** เพื่อคำนวณการซูมที่เหมาะสมที่สุดสำหรับความกว้างหน้าจอของคุณ

[image: quote-pdf-zoom — ภาพหน้าจอแสดงเอกสารที่ระดับการซูมต่าง ๆ หรือตัวควบคุมการซูมที่เน้น]

### เค้าโครงเอกสารใบเสนอราคา (Quote Document Layout)

เอกสาร PDF ที่แสดงผลเป็นรูปแบบ A4 แบบมืออาชีพ:

1. **ส่วนหัวบริษัท** — โลโก้บริษัท ชื่อ ที่อยู่ ข้อมูลติดต่อ
2. **ส่วนหัวใบเสนอราคา** — รหัสใบเสนอราคา วันที่ ใช้ได้ถึง ข้อมูลลูกค้า
3. **ที่อยู่ลูกค้า** — ที่อยู่จัดส่งและที่อยู่ออกใบแจ้งหนี้เรียงคู่กัน
4. **ตารางรายการสินค้า** — ชื่อชิ้นส่วน วัสดุ จำนวน ราคาต่อหน่วย ยอดรวมต่อรายการ
5. **ส่วนยอดรวม** — ยอดย่อย ค่าจัดส่ง VAT ยอดรวมทั้งหมด
6. **ส่วนเงื่อนไข** — ระยะเวลาการผลิต วิธีการจัดส่ง เงื่อนไขการชำระเงิน เงื่อนไขการค้า
7. **ส่วนท้าย** — รายละเอียดบริษัทและข้อมูลหน้า

[image: quote-pdf-document — ภาพหน้าจอของเอกสารใบเสนอราคาที่แสดงผลแสดงเค้าโครงทั้งหมดพร้อมส่วนหัวบริษัท ตารางรายการสินค้า และยอดรวม]

## การดาวน์โหลด PDF

1. ใน PDF Viewer คลิกปุ่ม **Download** (⬇)
2. ระบบจะสร้างไฟล์ PDF
3. ไฟล์จะดาวน์โหลดโดยอัตโนมัติโดยใช้ชื่อไฟล์ตรงกับรหัสใบเสนอราคา (เช่น `QT-2026-003.pdf`)

[image: quote-pdf-download — ภาพหน้าจอแสดงปุ่มดาวน์โหลดและ/หรือการแจ้งเตือนการดาวน์โหลดของเบราว์เซอร์]

---

## การพิมพ์ใบเสนอราคา

1. ใน PDF Viewer คลิกปุ่ม **Print** (🖨) หรือกด **⌘P** (Ctrl+P)
2. กล่องโต้ตอบพิมพ์ของเบราว์เซอร์จะเปิดขึ้น
3. เค้าโครงพิมพ์ถูกปรับให้เหมาะสมสำหรับกระดาษ A4 พร้อมระยะขอบที่เหมาะสม
4. เลือกเครื่องพิมพ์ของคุณแล้วคลิก Print

> **เคล็ดลับ:** สไตล์ชีทพิมพ์ (print stylesheet) จะซ่อนแถบเครื่องมือและการนำทาง — เฉพาะเอกสารใบเสนอราคาเท่านั้นที่จะถูกพิมพ์

---

## การแชร์ใบเสนอราคาผ่านลิงก์สาธารณะ (Public Link)

### การสร้างลิงก์แชร์

1. จากหน้า Quote Detail หรือ PDF Viewer คลิก **Share Link** (🔗)
2. ระบบจะสร้าง share token ที่ไม่ซ้ำกัน
3. URL สาธารณะจะถูกคัดลอกไปยังคลิปบอร์ดโดยอัตโนมัติ
4. การแจ้งเตือนยืนยัน: "Share link copied to clipboard." (คัดลอกลิงก์แชร์ไปยังคลิปบอร์ดแล้ว)

[image: quote-share-link-toast — ภาพหน้าจอแสดงการแจ้งเตือนความสำเร็จ "Share link copied to clipboard"]

### รูปแบบลิงก์แชร์ (Share Link Format)

URL แชร์มีรูปแบบดังนี้:
```
https://your-domain.com/share/quote/abc123-token-xyz
```

### สิ่งที่ลูกค้าเห็น

เมื่อลูกค้าเปิดลิงก์แชร์:

1. หน้าจะโหลดโดยไม่ต้องเข้าสู่ระบบ (เข้าถึงแบบสาธารณะ)
2. เอกสารใบเสนอราคาจะแสดงเป็น PDF ในเบราว์เซอร์
3. ลูกค้าสามารถดู เลื่อน และใช้ตัวควบคุม PDF ในตัวของเบราว์เซอร์เพื่อพิมพ์หรือบันทึกได้

[image: quote-share-customer-view — ภาพหน้าจอของหน้าแชร์สาธารณะที่ลูกค้าเห็น — PDF ฝังในพื้นหลังสีเทาเข้ม]

> **หมายเหตุ:** ลิงก์แชร์เป็นแบบ token-based เฉพาะคนที่มี URL นี้เท่านั้นที่สามารถเข้าถึงใบเสนอราคาได้ ลิงก์จะยังคงใช้ได้จนกว่า token จะถูกเพิกถอนหรือใบเสนอราคาถูกลบ
