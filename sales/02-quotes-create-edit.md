# 2. Quotes — สร้างและแก้ไข

## การสร้างใบเสนอราคาใหม่

1. ไปที่ **Sales → Quotes**
2. คลิกปุ่ม **+ Quote** ในแถบเครื่องมือ
3. แบบฟอร์มสร้างใบเสนอราคาจะเปิดที่ `/sales/quotes/create`

[image: quote-create-page — ภาพหน้าจอเต็มของแบบฟอร์ม Create Quote ว่างเปล่า แสดงฟิลด์ส่วนหัวและพื้นที่วางไฟล์แบบเขียนแบบ]

---

## แบบฟอร์มใบเสนอราคา — ฟิลด์ส่วนหัว (Header Fields)

ส่วนบนของแบบฟอร์มมีฟิลด์ระดับใบเสนอราคา:

| ฟิลด์ | จำเป็น | คำอธิบาย |
|-------|--------|---------|
| Quote Code (รหัสใบเสนอราคา) | ใช่ | สร้างอัตโนมัติ (เช่น `QT-2026-004`) สามารถแก้ไขด้วยตนเองได้ |
| Customer (ลูกค้า) | ใช่ | เมนูดรอปดาวน์เลือกจากลูกค้าที่มีอยู่ในโมดูล Knowledge |
| Created Date (วันที่สร้าง) | ใช่ | ค่าเริ่มต้นเป็นวันนี้ ตัวเลือกวันที่ (Date picker) |
| Valid Until (ใช้ได้ถึง) | ไม่ | วันหมดอายุ ต้องเป็นวันที่เท่ากับหรือหลัง Created Date |
| Lead Time (days) (ระยะเวลาการผลิต - วัน) | ไม่ | จำนวนวันทำการสำหรับการจัดส่งเริ่มต้น ค่าเริ่มต้นคือ 15 |
| Shipping Method (วิธีการจัดส่ง) | ใช่ | ค่าเริ่มต้นเป็น "DHL" พิมพ์ข้อความได้อิสระ |
| Shipping Cost (ค่าจัดส่ง) | ไม่ | ตัวเลข ค่าเริ่มต้นเป็น 0 |
| Payment Term (เงื่อนไขการชำระเงิน) | ใช่ | ค่าเริ่มต้นเป็น "Full payment in advance" (ชำระเงินล่วงหน้าเต็มจำนวน) พิมพ์ข้อความได้อิสระ |
| Trade Term (เงื่อนไขการค้า) | ใช่ | ค่าเริ่มต้นเป็น "EXW (Ex Works)" พิมพ์ข้อความได้อิสระ |
| Include Tax (VAT 7%) (รวมภาษี VAT 7%) | สวิตช์เปิด/ปิด | เมื่อเปิด จะเพิ่ม VAT 7% ในยอดรวม |
| Notes (หมายเหตุ) | ไม่ | พิมพ์ข้อความได้อิสระสำหรับหมายเหตุใบเสนอราคา |

[image: quote-form-header — ภาพระยะใกล้ของส่วนฟิลด์ส่วนหัวแสดง Customer dropdown, วันที่, lead time, การจัดส่ง และฟิลด์การชำระเงิน]

## การเพิ่มรายการสินค้า — ลากและวางแบบเขียนแบบ (Drag & Drop Drawings)

วิธีที่เร็วที่สุดในการเพิ่มรายการสินค้าคือการลากไฟล์แบบเขียนแบบวิศวกรรม:

1. ค้นหา **พื้นที่วาง** (drop zone) ด้านล่างฟิลด์ส่วนหัว จะแสดงข้อความ "Drop engineering drawings here" พร้อมเส้นขอบประ
2. ลากไฟล์หนึ่งหรือหลายไฟล์ (PDF, TIFF, PNG, JPG, STEP) จากคอมพิวเตอร์ของคุณไปยังพื้นที่วาง
3. แต่ละไฟล์จะสร้างรายการสินค้าใหม่โดยอัตโนมัติ:
   - **หมายเลขแบบเขียนแบบ** (drawing number) จะถูกดึงจากชื่อไฟล์
   - **ภาพขนาดย่อ** (thumbnail preview) จะถูกสร้างขึ้น
   - สำหรับไฟล์ STEP จะสร้าง **มุมมอง 3D แบบ isometric** โดยอัตโนมัติ

[image: quote-drawing-dropzone — ภาพหน้าจอของพื้นที่วางพร้อมเส้นขอบประและข้อความคำแนะนำ]

[image: quote-after-drop — ภาพหน้าจอแสดงรายการสินค้า 2–3 รายการที่สร้างอัตโนมัติหลังวางไฟล์แบบเขียนแบบ พร้อมภาพขนาดย่อ]

> **เคล็ดลับ:** คุณยังสามารถคลิก **+ Add Item** เพื่อเพิ่มรายการสินค้าเปล่าด้วยตนเองโดยไม่ต้องมีไฟล์แบบเขียนแบบ

## ฟิลด์รายการสินค้า (Line Item Fields)

แต่ละรายการสินค้ามีฟิลด์ดังนี้:

| ฟิลด์ | จำเป็น | คำอธิบาย |
|-------|--------|---------|
| Part/Product Name (ชื่อชิ้นส่วน/ผลิตภัณฑ์) | ใช่ | ชื่อของชิ้นส่วนที่เสนอราคา |
| Product Code (รหัสผลิตภัณฑ์) | ไม่ | หมายเลขชิ้นส่วน (ถ้ามี) |
| Drawing Number (หมายเลขแบบเขียนแบบ) | ไม่ | เติมอัตโนมัติจากชื่อไฟล์หากลากวาง หรือป้อนด้วยตนเอง |
| Material Spec (ข้อกำหนดวัสดุ) | ไม่ | ข้อกำหนดวัสดุ (เช่น "Al 6061-T6", "SS 304") |
| Surface Finish (การตกแต่งผิว) | ไม่ | การปรับสภาพผิว (เช่น "Anodized", "Powder Coat") |
| Quantity (จำนวน) | ใช่ | ต้องเป็นจำนวนเต็มบวก |
| Unit (หน่วย) | ใช่ | ค่าเริ่มต้นเป็น "pcs" (ชิ้น) |
| Unit Price (ราคาต่อหน่วย) | ใช่ | ราคาต่อหน่วย ต้อง ≥ 0 |
| Lead Time (days) (ระยะเวลาการผลิต - วัน) | ไม่ | ระยะเวลาการผลิตต่อรายการเป็นวันทำการ |
| Notes (หมายเหตุ) | ไม่ | หมายเหตุเฉพาะรายการ |

[image: quote-line-item-fields — ภาพระยะใกล้ของการ์ดรายการสินค้าเดียวแสดงฟิลด์ทั้งหมดพร้อมข้อมูลตัวอย่าง]

## กระบวนการผลิตต่อรายการ (Manufacturing Processes per Item)

แต่ละรายการสินค้าสามารถมีลำดับกระบวนการผลิตพร้อมต้นทุน:

1. ในการ์ดรายการสินค้า ค้นหาส่วน **Manufacturing Processes** ทางด้านขวา
2. คลิก **+ Add Process** เพื่อเพิ่มขั้นตอนกระบวนการ
3. สำหรับแต่ละกระบวนการ:
   - เลือก **Process Definition** (คำจำกัดความกระบวนการ) จากเมนูดรอปดาวน์ (เช่น "CNC Milling", "Deburring", "Surface Treatment")
   - หมายเลข **Sequence** (ลำดับ) จะถูกกำหนดโดยอัตโนมัติ (1, 2, 3…)
   - ป้อน **Cost per Unit** (ต้นทุนต่อหน่วย) สำหรับขั้นตอนกระบวนการนี้
   - เพิ่ม **Notes** (หมายเหตุ) ได้ตามต้องการ
4. ลากที่จับเพื่อ **จัดลำดับใหม่** ของขั้นตอนกระบวนการ
5. คลิกไอคอนถังขยะเพื่อ **ลบ** ขั้นตอนกระบวนการ

[image: quote-item-processes — ภาพระยะใกล้ของส่วน Manufacturing Processes แสดง 3 ขั้นตอนกระบวนการ (เช่น CNC Milling → Deburring → Anodizing) พร้อมหมายเลขลำดับและต้นทุน]

### โหลดจาก Process Route (Load from Process Route)

แทนที่จะเพิ่มกระบวนการด้วยตนเอง คุณสามารถโหลดจาก Process Route (เส้นทางกระบวนการ) ที่กำหนดไว้ล่วงหน้า:

1. คลิก **Load from Route** ที่ด้านบนของส่วนกระบวนการ
2. เลือก Process Route จากเมนูดรอปดาวน์ (เช่น "Standard CNC Route")
3. ขั้นตอนทั้งหมดจาก Route จะถูกเติมโดยอัตโนมัติพร้อมลำดับและต้นทุนเริ่มต้น

[image: quote-load-from-route — ภาพหน้าจอแสดงเมนูดรอปดาวน์ "Load from Route" ขยายออกพร้อม Process Route ที่มีให้เลือก]

## ภาพขนาดย่อของแบบเขียนแบบและมุมมอง Isometric (Drawing Thumbnails & Isometric Views)

เมื่อคุณวางไฟล์แบบเขียนแบบ 2D (PDF หรือ TIFF):

- **ภาพขนาดย่อ** จะถูกสร้างและแสดงในการ์ดรายการสินค้า
- หากระบบสามารถดึงมุมมอง isometric ได้ จะแสดงเป็นภาพขนาดย่อหลัก
- คลิกภาพขนาดย่อเพื่อเปิด **กล่องโต้ตอบดูภาพขนาดเต็ม** ที่แสดงแบบเขียนแบบทั้งหมด

[image: quote-drawing-fullsize — ภาพหน้าจอของกล่องโต้ตอบดูแบบเขียนแบบขนาดเต็มที่เปิดเมื่อคลิกภาพขนาดย่อ]

## Delivery Promise Signal (สัญญาณการจัดส่งตามกำหนด) — ต่อรายการ

หากรายการสินค้ามีกระบวนการผลิตและกำหนด lead time ไว้ **Delivery Promise Signal** จะปรากฏด้านล่างรายละเอียดรายการ:

- 🟢 **เขียว** — เป็นไปได้: มีกำลังการผลิตเพียงพอพร้อมวันสำรอง (slack days)
- 🟡 **เหลือง** — แน่น: กำลังการผลิตใกล้ถึงขีดจำกัดหรือวันสำรองน้อยมาก
- 🔴 **แดง** — มีความเสี่ยง: กำลังการผลิตไม่เพียงพอที่จะตอบสนอง lead time

[image: quote-item-delivery-signal — ภาพระยะใกล้ของรายการสินค้าแสดงตัวบ่งชี้ delivery promise สีเขียว/เหลือง/แดง ด้านล่างฟิลด์]

ดู [Delivery Promise Signal (สัญญาณการจัดส่งตามกำหนด)](./09-delivery-promise.md) สำหรับรายละเอียดทั้งหมด

## Aggregate Delivery Signal (สัญญาณการจัดส่งรวม)

ที่ด้านบนของแบบฟอร์ม (ใกล้ส่วนหัว) **aggregate delivery signal** (สัญญาณการจัดส่งรวม) จะแสดงความเป็นไปได้โดยรวมของทุกรายการสินค้า ช่วยให้คุณประเมินได้อย่างรวดเร็วก่อนส่งใบเสนอราคา

[image: quote-aggregate-signal — ภาพหน้าจอของ aggregate delivery promise signal ใกล้ด้านบนของแบบฟอร์ม]

## ยอดรวมใบเสนอราคา (Quote Totals)

ที่ด้านล่างของแบบฟอร์ม ระบบจะคำนวณ:

| รายการ | การคำนวณ |
|--------|---------|
| Subtotal (ยอดย่อย) | ผลรวมของ (unit_price × quantity) สำหรับทุกรายการ |
| Shipping (ค่าจัดส่ง) | จากฟิลด์ Shipping Cost |
| VAT 7% (ภาษีมูลค่าเพิ่ม 7%) | (Subtotal + Shipping) × 0.07 — เฉพาะเมื่อเปิด "Include Tax" |
| **Total** (ยอดรวมทั้งหมด) | Subtotal + Shipping + VAT |

[image: quote-totals — ภาพระยะใกล้ของส่วนยอดรวมด้านล่างแสดง Subtotal, Shipping, VAT และ Total]

## การบันทึกใบเสนอราคา

1. ตรวจสอบฟิลด์และรายการสินค้าทั้งหมด
2. คลิก **Save** (หรือ **Create Quote** สำหรับใบเสนอราคาใหม่)
3. ใบเสนอราคาจะถูกบันทึกในสถานะ **Draft**
4. คุณจะถูกนำไปยังหน้ารายละเอียดใบเสนอราคา (Quote Detail)
5. การแจ้งเตือนความสำเร็จจะปรากฏ: "Quote created successfully." (สร้างใบเสนอราคาสำเร็จ)

[image: quote-save-success — ภาพหน้าจอแสดงการแจ้งเตือนหลังบันทึก]

---

## การแก้ไขใบเสนอราคาที่มีอยู่

1. จากรายการใบเสนอราคา คลิก **ไอคอนดินสอ** (✏) ที่แถวใบเสนอราคา Draft
   - หรือจากหน้ารายละเอียดใบเสนอราคา คลิกปุ่ม **Edit**
2. แบบฟอร์มแก้ไขใบเสนอราคาจะโหลดที่ `/sales/quotes/edit/:id` พร้อมข้อมูลที่มีอยู่ทั้งหมด
3. ทำการเปลี่ยนแปลงที่ต้องการ
4. คลิก **Save** เพื่ออัปเดต

> **หมายเหตุ:** เฉพาะใบเสนอราคาที่มีสถานะ **Draft** หรือ **Rejected** เท่านั้นที่สามารถแก้ไขได้ ใบเสนอราคาสถานะ Sent, Accepted, Expired และ Cancelled เป็นแบบอ่านอย่างเดียว

[image: quote-edit-button — ภาพหน้าจอแสดงตำแหน่งปุ่ม Edit บนส่วนหัวหน้ารายละเอียดใบเสนอราคา]
