# 5. Scheduling Board — ภาพรวมและเลย์เอาต์

## การเข้าถึง Scheduling Board

นำทางไปที่ **Production → Scheduling** หรือไปที่ `/production/scheduling`

นี่คือเครื่องมือหลักของผู้วางแผน — อินเทอร์เฟซการจัดตารางแบบภาพสามเลน

[image: scheduling-board-full — ภาพหน้าจอเต็มของหน้า Scheduling Board แสดงทั้งสามเลน: Timeline Grid, Outside Processing Panel, และ QC Queue Panel]

---

## เลย์เอาต์ของหน้า (Page Layout)

Scheduling Board มีพื้นที่หลักดังนี้:

### แถบเครื่องมือด้านบน (Top Toolbar)

| ตัวควบคุม | คำอธิบาย |
|---------|-------------|
| View toggle (สลับมุมมอง) | สลับระหว่าง **Machine View** (มุมมองเครื่องจักร) และ **Line Item View** (มุมมองรายการ) |
| Week navigator (ตัวนำทางสัปดาห์) | ← สัปดาห์ก่อน / วันนี้ / สัปดาห์ถัดไป → |
| ⚡ Generate Schedule (สร้างตาราง) | เปิดกล่องโต้ตอบการจัดตารางอัตโนมัติ |
| Zoom controls (ตัวควบคุมการซูม) | ปรับระดับการซูมของ Timeline |

[image: scheduling-toolbar — ภาพขยายของแถบเครื่องมือแสดง View toggle, Week navigator, และปุ่ม Generate Schedule]

### แถบสรุปกำลังการผลิต (Capacity Summary Strip)

ใต้แถบเครื่องมือ แถบข้อมูลขนาดกะทัดรัดแสดง:
- ชั่วโมงที่พร้อมใช้งานทั้งหมดของสัปดาห์
- ชั่วโมงที่จัดตารางแล้ว
- เปอร์เซ็นต์การใช้งานพร้อมแถบแสดงผล
- ชั่วโมงทำงานล่วงเวลา (ถ้ามี)

[image: scheduling-capacity-strip — ภาพขยายของแถบสรุปกำลังการผลิต]

## เลย์เอาต์สามเลน (Three-Lane Layout)

กระดานถูกจัดวางแนวตั้งเรียงซ้อนกันสามเลนที่จัดแนวเวลาตรงกัน:

```
┌─────────────────────────────────────────────────────────┐
│  LANE 1: Timeline Grid (กริดตารางเวลา)                   │
│  แถวอุปกรณ์ × คอลัมน์ชั่วโมง (08:00–20:00)               │
│  แถบงานที่ลากได้ต่อเครื่องจักรต่อวัน                        │
├─────────────────────────────────────────────────────────┤
│  LANE 2: Outside Processing Panel (แผงงานส่งผลิตภายนอก)  │
│  แถวผู้ขาย × คอลัมน์วัน (แถบ Gantt)                       │
│  สี: ส้ม=จัดส่งแล้ว, แดง=ล่าช้า, เทา=รอดำเนินการ          │
├─────────────────────────────────────────────────────────┤
│  LANE 3: QC Queue Panel (แผงคิว QC)                      │
│  คอลัมน์จัดกลุ่มตามวันพร้อมการ์ดการตรวจสอบ                │
│  ตัวบ่งชี้สีแดงสำหรับรายการเกินกำหนด                       │
└─────────────────────────────────────────────────────────┘
```

[image: scheduling-three-lanes — ภาพหน้าจอพร้อมคำอธิบายแสดงทั้งสามเลนที่มีลูกศรระบุ]

---

## Lane 1: Timeline Grid (กริดตารางเวลา)

พื้นที่จัดตารางหลัก แต่ละแถวแทนเครื่องจักร แต่ละคอลัมน์แทนหนึ่งชั่วโมงของวัน

### แกนเวลา (Time Axis)

- **ชั่วโมงทำงาน:** 08:00–20:00 (12 ชั่วโมงต่อวัน)
- **โซนเวลา:**
  - ปกติ: 08:00–11:00 และ 13:00–17:00 (พื้นหลังสีขาว)
  - พักเบรก: 12:00–13:00 (แถบสีเทาอ่อน)
  - ทำงานล่วงเวลา: 18:00–20:00 (แถบสีอำพันอ่อน)
- **วันหยุด**จะถูกบีบอัด (คอลัมน์แคบกว่า พื้นหลังเทา)
- **วันนี้**จะถูกเน้นด้วยสีน้ำเงินอ่อน

[image: scheduling-timeline-axis — ภาพขยายของแกนเวลาแสดงโซนปกติ พักเบรก และทำงานล่วงเวลาพร้อมสี]

### แถวอุปกรณ์ (Equipment Rows)

แต่ละแถวแสดง:
- ชื่อเครื่องจักรและรหัสที่ป้ายด้านซ้าย
- แถบงานที่จัดตำแหน่งตาม start_hour และระยะเวลา
- ช่องว่างที่สามารถวางงานได้

[image: scheduling-equipment-rows — ภาพขยายของ 3-4 แถวอุปกรณ์พร้อมแถบงานหลากสีและขนาด]

### แถบงาน (Operation Bars)

แต่ละแถบแทนงานที่จัดตารางแล้ว (ขั้นตอน WO ที่มอบหมายให้เครื่องจักรในเวลาที่กำหนด):

| องค์ประกอบภาพ | ความหมาย |
|----------------|---------|
| สีแถบ | ตามความสำคัญของ WO (แดง=urgent, ส้ม=high, น้ำเงิน=normal, เทา=low) |
| ความกว้างแถบ | เป็นสัดส่วนกับชั่วโมงที่ประมาณ |
| ป้ายรหัส WO | แสดงบนแถบ (เช่น "WO-2026-00042") |
| ชื่อขั้นตอน | แสดงใต้รหัส (เช่น "CNC Milling") |
| จุดสถานะ | เขียว=completed, น้ำเงิน=in_progress, เทา=pending |

[image: scheduling-operation-bar — ภาพขยายของแถบงานเดียวแสดงรหัส WO, ชื่อขั้นตอน, และตัวบ่งชี้สถานะ]

### การโต้ตอบกับแถบงาน

- **วางเมาส์ (Hover)** — แสดงการ์ดข้อมูล WO พร้อมรายละเอียดครบถ้วน (ขั้นตอน, ความสำคัญ, วันกำหนด, ความคืบหน้า, รูป Isometric)
- **คลิก (Click)** — เปิดแผ่นรายละเอียดงาน (Operation Detail Sheet) (ดูด้านล่าง)
- **ลาก (Drag)** — ย้ายงานไปยังเครื่องจักรหรือช่วงเวลาอื่น

[image: scheduling-wo-hover-card — ภาพหน้าจอของการ์ดข้อมูล WO ที่ปรากฏเมื่อวางเมาส์เหนือแถบงาน]

## Lane 2: Outside Processing Panel (แผงงานส่งผลิตภายนอก)

แสดงงานผู้ขายเป็นแถบ Gantt ที่จัดแนวตามคอลัมน์วันเดียวกับ Timeline

- **ยุบ/ขยายได้** — คลิกที่ส่วนหัวเพื่อยุบ/ขยาย
- **ส่วนหัว**แสดงจำนวนสรุป (เช่น "3 active, 1 late")
- แต่ละแถวแทนผู้ขาย
- แถบครอบคลุมตั้งแต่วันจัดส่งถึงวันที่คาดว่าจะส่งคืน
- รหัสสี: เทา=pending, ส้ม=shipped, แดง=late

[image: scheduling-opo-panel — ภาพหน้าจอของ Outside Processing Panel แสดงแถวผู้ขายพร้อมแถบ Gantt]

ดู [Outside Processing Panel](./08-outside-processing-panel.md) สำหรับรายละเอียด

## Lane 3: QC Queue Panel (แผงคิว QC)

แสดงขั้นตอนตรวจสอบคุณภาพที่กำลังจะมาถึงจัดกลุ่มตามวัน

- **ยุบ/ขยายได้** — คลิกที่ส่วนหัวเพื่อยุบ/ขยาย
- **ส่วนหัว**แสดงจำนวนที่รอ (เช่น "5 pending")
- การ์ดจัดกลุ่มตามคอลัมน์วัน
- แต่ละการ์ดแสดงรหัส WO, ชื่อผลิตภัณฑ์, ชื่อขั้นตอน QC
- ตัวบ่งชี้สีแดงสำหรับรายการเกินกำหนด
- ป้ายสถานะขั้นตอนก่อนหน้า (Ready, In Progress, Pending)

[image: scheduling-qc-panel — ภาพหน้าจอของ QC Queue Panel แสดงการ์ดจัดกลุ่มตามวัน]

ดู [QC Queue Panel](./09-qc-queue-panel.md) สำหรับรายละเอียด

## การเชื่อมโยงข้ามเลนเมื่อวางเมาส์ (Cross-Lane Hover Linking)

เมื่อคุณวางเมาส์เหนือ WO ใดก็ได้ในเลนใดก็ได้ ระบบจะเน้น WO นั้นข้ามทั้งสามเลน:

1. WO ที่ถูกวางเมาส์จะได้ **วงแหวนสีน้ำเงิน** (ring-2 blue) ในทุกเลนที่ปรากฏ
2. รายการอื่นทั้งหมดจะ**จางลงเหลือ 35% opacity**
3. **FlowIndicator** จะแสดงจุด breadcrumb ที่ขอบซ้ายของแต่ละเลนที่ WO ปรากฏ เชื่อมต่อด้วยเส้นแนวตั้ง

สิ่งนี้ช่วยให้คุณเห็นการไหลของใบสั่งผลิตอย่างครบถ้วนทันที: อยู่บนเครื่องจักรใด มีงานส่งผลิตภายนอกหรือไม่ และจะถึงขั้นตอน QC เมื่อไร

[image: scheduling-hover-linking — ภาพหน้าจอแสดง WO ที่ถูกเน้นข้ามทั้งสามเลนพร้อมจุด FlowIndicator ที่มองเห็นได้]

## แถบด้านข้างงานที่ยังไม่จัดตาราง (Unscheduled Sidebar)

ทางด้านซ้ายของกระดาน แถบด้านข้างแสดงรายการงานทั้งหมดที่ยังไม่จัดตารางจัดกลุ่มตามใบสั่งผลิต

### การ์ดกลุ่ม WO (WO Group Card)

แต่ละกลุ่มแสดง:
- รหัส WO (monospace, ตัวหนา)
- ป้ายความสำคัญ (urgent=แดง "!", high=ส้ม "H")
- ชื่อผลิตภัณฑ์และชื่อลูกค้า
- แถบความคืบหน้า (จัดตารางแล้ว/ขั้นตอนทั้งหมด)
- จำนวนที่ยังไม่จัดตารางและชั่วโมงรวม
- วันกำหนดสัมพัทธ์ (เช่น "2d overdue", "due tomorrow", "due in 5d")
- รูปชิ้นส่วนขนาดย่อ

[image: scheduling-unscheduled-sidebar — ภาพหน้าจอของแถบด้านข้างงานที่ยังไม่จัดตารางแสดง 3-4 กลุ่ม WO พร้อมความสำคัญและวันกำหนดที่แตกต่างกัน]

### การ์ดขั้นตอน (Step Cards) (ภายในกลุ่ม)

คลิกกลุ่ม WO เพื่อขยายและดูขั้นตอนที่ยังไม่จัดตารางแต่ละรายการ:

| องค์ประกอบ | คำอธิบาย |
|---------|-------------|
| Grip handle (ที่จับลาก) | ลากเพื่อจัดตารางบน Timeline |
| Predecessor dot (จุดขั้นตอนก่อนหน้า) | เขียว = ขั้นตอนก่อนหน้าเสร็จ (พร้อม), เทา = ขั้นตอนก่อนหน้ายังไม่เสร็จ |
| Sequence number (หมายเลขลำดับ) | ลำดับขั้นตอนภายใน WO |
| Step name (ชื่อขั้นตอน) | ชื่อกระบวนการ (เช่น "CNC Milling") |
| Estimated hours (ชั่วโมงประมาณ) | ระยะเวลา (เช่น "4h") |
| Equipment type (ประเภทอุปกรณ์) | ประเภทเครื่องจักรที่ต้องการ (ข้อความเล็กด้านล่าง) |

[image: scheduling-unscheduled-steps — ภาพขยายของการ์ดขั้นตอนที่ขยายภายในกลุ่ม WO แสดงที่จับลากและจุดขั้นตอนก่อนหน้า]

### ขั้นตอนงานส่งผลิตภายนอกในแถบด้านข้าง

ขั้นตอน OPO ปรากฏเป็นแถวข้อมูลที่ลากไม่ได้พร้อมไอคอน 🚚 รถบรรทุกและระยะเวลานำส่งของผู้ขาย ไม่สามารถจัดตารางบนเครื่องจักรได้ — แสดงช่วงเวลาว่าง

[image: scheduling-unscheduled-opo-step — ภาพขยายของขั้นตอน OPO ในแถบด้านข้างแสดงไอคอนรถบรรทุกและระยะเวลานำส่ง]

## แผ่นรายละเอียดงาน (Operation Detail Sheet)

คลิกแถบงานที่จัดตารางแล้วเพื่อเปิดแผ่นรายละเอียดจากด้านขวา:

| ส่วน | เนื้อหา |
|---------|---------|
| Header (ส่วนหัว) | รหัส WO, ชื่อขั้นตอน, ป้ายความสำคัญ |
| Schedule (ตาราง) | ชื่อเครื่องจักร, วันที่, ชั่วโมงเริ่มต้น, ชั่วโมงประมาณ |
| Plan vs Actual (แผนเทียบกับจริง) | ระยะเวลาที่วางแผนเทียบกับชั่วโมงจริง, ตัวบ่งชี้เกินเวลา |
| Status (สถานะ) | ป้ายสถานะขั้นตอน, การตรวจจับเกินเวลา |
| Actions (การดำเนินการ) | **Edit** (แก้ไขเครื่องจักร/เวลา), **Unschedule** (ลบออกจาก Timeline) |

[image: scheduling-operation-detail — ภาพหน้าจอของแผ่นรายละเอียดงานแสดงทุกส่วน]

## โหมดมุมมอง (View Modes)

### Machine View (มุมมองเครื่องจักร) (ค่าเริ่มต้น)

แถวอุปกรณ์ × คอลัมน์ชั่วโมง เหมาะที่สุดสำหรับดูการใช้งานเครื่องจักรและจัดตารางงาน

### Line Item View (มุมมองรายการ)

จัดกลุ่มงานตาม Sales Order Line Item แทนที่จะจัดตามเครื่องจักร แสดงการไหลของแต่ละชิ้นส่วนผ่านขั้นตอนกระบวนการข้ามเครื่องจักรและวัน

สลับระหว่างมุมมองโดยใช้แท็บในแถบเครื่องมือ

[image: scheduling-line-item-view — ภาพหน้าจอของ Line Item View แสดงงานจัดกลุ่มตาม SO Line Item]
