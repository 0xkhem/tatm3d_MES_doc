# 2. Production Dashboard (แดชบอร์ดการผลิต)

## การเข้าถึง Dashboard

นำทางไปที่ **Production → Dashboard** หรือไปที่ `/production/dashboard`

[image: production-dashboard-full — ภาพหน้าจอเต็มของหน้า Production Dashboard แสดงการ์ด KPI, แถบการกระจายสถานะ, ตาราง WO, และ Process Tasks]

---

## การ์ด KPI (KPI Cards)

แถวบนสุดแสดงตัวชี้วัดการผลิตหลัก:

| KPI | คำอธิบาย |
|-----|-------------|
| Active Projects (โครงการที่ดำเนินการอยู่) | โครงการที่กำลังดำเนินการอยู่ |
| WIP Count (จำนวนงานระหว่างผลิต) | ใบสั่งผลิตที่มีสถานะ "in_progress" |
| On-Time Rate (อัตราตรงเวลา) | เปอร์เซ็นต์ของ WO ที่เสร็จภายในวันกำหนด |
| Quality Pass Rate (อัตราผ่านคุณภาพ) | เปอร์เซ็นต์ของการตรวจสอบที่ผ่าน |
| Overdue WOs (WO เกินกำหนด) | จำนวน WO ที่เกินวันกำหนด (แจ้งเตือนสีแดงถ้า > 0) |

[image: dashboard-kpi-cards — ภาพขยายของแถวการ์ด KPI พร้อมค่าตัวอย่าง]

---

## แถบการกระจายสถานะ (Status Distribution Bar)

แถบแนวนอนแบบซ้อนแสดงการกระจายของใบสั่งผลิตทั้งหมดตามสถานะ:

| สถานะ | สี |
|--------|-------|
| Draft (ร่าง) | เทาหินชนวน |
| Released (ปล่อยแล้ว) | น้ำเงิน |
| In Progress (กำลังดำเนินการ) | อำพัน |
| On Hold (พักงาน) | แดง |
| Completed (เสร็จสิ้น) | เขียว |
| Cancelled (ยกเลิก) | เทา |

วางเมาส์เหนือส่วนใดส่วนหนึ่งเพื่อดูจำนวนและเปอร์เซ็นต์ที่แน่นอน

[image: dashboard-status-bar — ภาพหน้าจอของแถบการกระจายสถานะพร้อม tooltip แสดงจำนวนเมื่อวางเมาส์]

---

## ตาราง Work Order (Work Order Table)

ใต้ KPI เป็นตารางที่แสดง Work Order ล่าสุดพร้อมข้อมูล:

- รหัสคำสั่ง (คลิกได้)
- ชื่อผลิตภัณฑ์และสเปค
- ชื่อลูกค้า
- ป้ายสถานะ (Status badge)
- ป้ายความสำคัญ (Priority badge)
- ความคืบหน้าของขั้นตอน (เสร็จ/ทั้งหมด)
- รูปภาพ Isometric ขนาดเล็ก (ถ้ามี)
- ป้าย Rework/Remake (ถ้ามี)

ใช้ checkbox เพื่อเลือก WO สำหรับการดำเนินการแบบกลุ่ม

[image: dashboard-wo-table — ภาพหน้าจอของตาราง Work Order แสดง 4-5 แถวพร้อมป้ายสถานะและตัวบ่งชี้ความคืบหน้า]

---

## งานกระบวนการ (Process Tasks)

ส่วนที่แสดงงานกระบวนการที่ดำเนินอยู่ทั่วพื้นที่ผลิต — ขั้นตอนไหนกำลังดำเนินการ อยู่บนเครื่องจักรใด และใช้เวลาไปแล้วเท่าไร

[image: dashboard-process-tasks — ภาพหน้าจอของส่วน Process Tasks]
