# 6. บอร์ดจัดตาราง — การกำหนดด้วยตนเอง

## สองวิธีในการจัดตารางด้วยตนเอง

### วิธี A: ลากและวาง

1. ใน **Unscheduled Sidebar** ขยายกลุ่ม WO เพื่อดูขั้นตอน
2. ตรวจสอบ **จุด Predecessor** — สีเขียวหมายความว่าขั้นตอนพร้อมจัดตาราง
3. จับ **ที่จับลาก** (⠿) บนการ์ดขั้นตอน
4. ลากไปยัง **Timeline Grid** — วางบนแถวเครื่องจักรและช่องเวลาที่ต้องการ
5. แถบการดำเนินงานปรากฏทันที (Optimistic Update)

[image: scheduling-drag-drop — ภาพหน้าจอลำดับแสดงขั้นตอนที่ถูกลากจาก Sidebar ไปยัง Timeline Grid]

> **หมายเหตุ:** เฉพาะขั้นตอนในบ้านเท่านั้นที่สามารถลากได้ ขั้นตอน Outside Processing (🚚) ไม่สามารถลากได้

### วิธี B: กล่องโต้ตอบกำหนด

1. ใน **Unscheduled Sidebar** คลิกการ์ดขั้นตอน (ไม่ใช่ที่จับลาก)
2. **กล่องโต้ตอบ Assign Operation** จะเปิดขึ้น
3. กรอกรายละเอียดการกำหนด (ดูด้านล่าง)
4. คลิก **Save**

[image: scheduling-assign-dialog — ภาพหน้าจอของกล่องโต้ตอบ Assign Operation]

---

## กล่องโต้ตอบ Assign Operation

| ฟิลด์ | คำอธิบาย |
|-------|-------------|
| WO Code | อ่านอย่างเดียว — แสดงรหัส Work Order |
| Step Name | อ่านอย่างเดียว — แสดงชื่อขั้นตอนกระบวนการ |
| Machine | Dropdown กรองตามประเภทเครื่องจักรที่ขั้นตอนต้องการ |
| Date | ตัวเลือกวันที่สำหรับวันที่จัดตาราง |
| Start Hour | Dropdown พร้อมช่วง 30 นาที (08:00, 08:30, 09:00, … 19:30) |
| Estimated Hours | ระยะเวลาเป็นชั่วโมง (กรอกล่วงหน้าจากการประมาณของขั้นตอน) |

### ขั้นตอนการกำหนด

1. เลือก **Machine** จาก Dropdown (แสดงเฉพาะเครื่องจักรที่ตรงกับประเภทเครื่องจักรที่ต้องการ)
2. เลือก **Date**
3. เลือก **Start Hour**
4. ปรับ **Estimated Hours** ถ้าจำเป็น
5. คลิก **Save**

[image: scheduling-assign-fields — ภาพใกล้ของฟิลด์กล่องโต้ตอบพร้อมค่าตัวอย่างที่กรอก]

---

## การตรวจจับล่วงเวลา

ถ้าการดำเนินงานขยายเกิน 18:00 (เข้าสู่โซนล่วงเวลา) **กล่องโต้ตอบยืนยันล่วงเวลา** จะปรากฏ:

| ตัวเลือก | สิ่งที่เกิดขึ้น |
|--------|-------------|
| **Use Overtime** | การดำเนินงานถูกจัดตารางในโซนล่วงเวลา 18:00–20:00 |
| **Push to Next Day** | ชั่วโมงที่เหลือจะล้นไปยังวันทำงานถัดไปที่ 08:00 |

กล่องโต้ตอบแสดงจำนวนชั่วโมงที่อยู่ในโซนล่วงเวลา (เช่น "1.5h will fall in the overtime zone")

[image: scheduling-ot-dialog — ภาพหน้าจอของกล่องโต้ตอบยืนยันล่วงเวลาแสดงสองตัวเลือก]

---

## การแก้ไขการดำเนินงานที่จัดตารางแล้ว

1. คลิกแถบการดำเนินงานบน Timeline เพื่อเปิด **Operation Detail Sheet**
2. คลิก **Edit** (✏)
3. กล่องโต้ตอบ Assign จะเปิดใหม่พร้อมค่าปัจจุบันที่กรอกไว้ล่วงหน้า
4. เปลี่ยนเครื่องจักร, วันที่ หรือเวลาตามต้องการ
5. คลิก **Save**

[image: scheduling-edit-operation — ภาพหน้าจอของขั้นตอนการแก้ไขจาก Detail Sheet]

---

## การยกเลิกการจัดตารางการดำเนินงาน

1. คลิกแถบการดำเนินงานเพื่อเปิด **Operation Detail Sheet**
2. คลิก **Unschedule** (🗑)
3. การดำเนินงานถูกลบออกจาก Timeline และกลับไปยัง Unscheduled Sidebar

[image: scheduling-unschedule — ภาพหน้าจอแสดงปุ่ม Unschedule ใน Detail Sheet]

---

## การย้ายการดำเนินงาน (ลากบน Timeline)

การดำเนินงานที่จัดตารางแล้วสามารถย้ายได้:

1. จับแถบการดำเนินงานบน Timeline
2. ลากไปยังแถวเครื่องจักรหรือช่องเวลาอื่น
3. วางในตำแหน่งใหม่
4. การอัปเดตถูกใช้แบบ Optimistic (ข้อเสนอแนะ UI ทันที, ซิงค์พื้นหลัง)

> **การตรวจจับความขัดแย้ง:** ถ้าตำแหน่งใหม่ทับซ้อนกับการดำเนินงานอื่นบนเครื่องจักรเดียวกัน ระบบจะเตือนคุณ

[image: scheduling-move-operation — ภาพหน้าจอแสดงการดำเนินงานที่ถูกลากไปยังตำแหน่งใหม่บน Timeline]

---

## เคล็ดลับสำหรับการจัดตารางด้วยตนเอง

| เคล็ดลับ | รายละเอียด |
|-----|---------|
| ตรวจสอบจุด Predecessor | จุดสีเขียว = พร้อมจัดตาราง จุดสีเทา = Predecessor ยังไม่เสร็จ — การจัดตารางตอนนี้หมายความว่าไม่สามารถเริ่มได้จนกว่า Predecessor จะเสร็จ |
| ดูแถบกำลังการผลิต | แถบ Utilization อัปเดตแบบเรียลไทม์เมื่อคุณเพิ่มการดำเนินงาน |
| ใช้ Hover Card | วางเมาส์บนแถบใดก็ได้เพื่อดูบริบท WO แบบเต็มก่อนตัดสินใจ |
| เคารพประเภทเครื่องจักร | กล่องโต้ตอบ Assign แสดงเฉพาะเครื่องจักรที่เข้ากันได้ แต่การลากและวางอนุญาตให้ใช้แถวใดก็ได้ — ตรวจสอบให้แน่ใจว่าคุณวางบนประเภทเครื่องจักรที่ถูกต้อง |
| พิจารณาล่วงเวลา | การดำเนินงานในโซน 18:00–20:00 เป็นล่วงเวลา — ใช้อย่างประหยัด |
