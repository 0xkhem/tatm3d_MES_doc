# Equipment (เครื่องจักร)

## คืออะไร?

โมดูล Equipment ติดตามสถานะและการซ่อมบำรุงของเครื่องจักรทุกเครื่องในหน้างาน ให้ผู้วางแผนและทีมซ่อมบำรุงมุมมอง live ของความพร้อมใช้งานของเครื่องจักร เพื่อให้การตัดสินใจจัดตารางอยู่บนพื้นฐานของความเป็นจริง

## Factory Floor Overview (ภาพรวมพื้นที่โรงงาน)

Factory Floor Overview คือแผนที่ภาพของโรงงาน เครื่องจักรถูกจัดกลุ่มตาม Workshop (เช่น "CNC Bay", "Grinding Section") และ Workstation แต่ละ Tile เครื่องจักรแสดงสถานะปัจจุบันได้ทันที

สถานะเครื่องจักร:

| สถานะ | สี | ความหมาย |
|-------|---|----------|
| Available | เขียว | พร้อมทำงาน |
| In Use | น้ำเงิน | กำลังรันงานอยู่ |
| Under Maintenance | เหลือง | กำลังซ่อมบำรุงตามแผน |
| Breakdown | แดง | หยุดทำงานโดยไม่ได้วางแผน |
| Offline | เทา | ปิดเครื่องหรือปลดระวาง |

## รายการเครื่องจักร

หน้ารายการเครื่องจักรแสดงเครื่องจักรทั้งหมดพร้อมรหัส ประเภท สถานะปัจจุบัน และวันซ่อมบำรุงล่าสุด สามารถกรองตามประเภทเครื่องจักรหรือสถานะ

รูปแบบรหัสเครื่องจักร: `EQP-0001`

## Maintenance Tasks (งานซ่อมบำรุง)

Maintenance Tasks คืองานซ่อมบำรุงตามแผนที่เกิดซ้ำ เช่น เปลี่ยนน้ำมัน, สอบเทียบ, เปลี่ยนไส้กรอง แต่ละงานมี:

- เครื่องจักรที่เกี่ยวข้อง
- ประเภทการซ่อมบำรุง (ป้องกัน, พยากรณ์ ฯลฯ)
- วันที่กำหนดและระยะเวลาประมาณ
- ช่างที่ได้รับมอบหมาย
- สถานะ (กำหนดไว้, กำลังดำเนินการ, เสร็จสมบูรณ์)

## Maintenance Requests (คำร้องขอซ่อมบำรุง)

Maintenance Requests คือคำร้องขอซ่อมแซมเฉพาะกิจที่ช่างปฏิบัติงานหรือช่างเทคนิคส่งเมื่อมีอะไรเสียหรือทำงานผิดปกติ คำร้องบันทึก:

- เครื่องจักรที่ได้รับผลกระทบ
- คำอธิบายปัญหา
- ความสำคัญ (ต่ำ, กลาง, สูง, วิกฤต)
- ผู้ส่งคำร้องและเวลา

คำร้องถูกตรวจสอบโดยทีมซ่อมบำรุงและแปลงเป็น Task เมื่อได้รับการอนุมัติ

## แนวคิดหลัก

| คำศัพท์ | ความหมาย |
|---------|----------|
| Equipment Type | หมวดหมู่เครื่องจักร (เช่น "5-Axis CNC Mill", "CMM") |
| Equipment Instance | เครื่องจักรจริงเฉพาะเครื่อง (เช่น "CNC-5A-01") |
| Workshop | พื้นที่หรือ Bay ที่จัดกลุ่มเครื่องจักร |
| Workstation | ตำแหน่งเฉพาะภายใน Workshop |
| Maintenance Task | งานซ่อมบำรุงตามแผน |
| Maintenance Request | คำร้องขอซ่อมแซมที่ไม่ได้วางแผน |
