# Tooling (เครื่องมือตัด)

## คืออะไร?

โมดูล Tooling จัดการวงจรชีวิตของเครื่องมือตัด CNC ตั้งแต่การกำหนดประเภทเครื่องมือที่มีอยู่ ไปจนถึงการติดตามเครื่องมือจริงแต่ละชิ้น และรู้ว่าเครื่องมือไหนถูกโหลดในเครื่องจักรไหนอยู่ตอนนี้ นอกจากนี้ยังแจ้งเตือนเมื่อเครื่องมือใกล้หมดอายุการใช้งาน

## ลำดับชั้นเอนทิตี

```
Tool Category → Tool Definition → Tool Instance → Machine Assignment
```

**Tool Category** — ป้ายจัดกลุ่ม (เช่น "End Mills", "Drills", "Inserts", "Boring Bars")

**Tool Definition** — เทมเพลตที่ใช้ซ้ำได้สำหรับประเภทเครื่องมือ กำหนดข้อมูลจำเพาะและขีดจำกัดอายุสูงสุด รูปแบบรหัส: `TLD-0001`

**Tool Instance** — เครื่องมือจริงในโรงงาน ติดตามว่าใช้อายุไปเท่าไหร่แล้ว รูปแบบรหัส: `TLI-0001`

**Machine Assignment** — บันทึกว่า Tool Instance ไหนถูกโหลดในเครื่องจักรไหน และอยู่ใน Magazine Pocket ไหน

## การติดตามอายุเครื่องมือ

แต่ละ Tool Definition สามารถมีขีดจำกัดอายุได้สูงสุดสามมิติ:

| มิติ | วัดอะไร |
|-----|---------|
| Max cuts | จำนวนครั้งการตัดทั้งหมด |
| Max minutes | เวลาตัดรวมเป็นนาที |
| Max parts | จำนวนชิ้นงานที่ผลิตทั้งหมด |

ระบบติดตามการใช้งานปัจจุบันเทียบกับแต่ละขีดจำกัดและคำนวณเปอร์เซ็นต์อายุ เปอร์เซ็นต์สูงสุดในทุกมิติกำหนดสถานะของเครื่องมือ:

| อายุ % | สถานะ | ความหมาย |
|--------|-------|----------|
| < 80% | OK | เครื่องมือยังดี |
| 80–99% | Warning | วางแผนเปลี่ยนเร็วๆ นี้ |
| ≥ 100% | Critical | เครื่องมือเกินขีดจำกัด — เปลี่ยนทันที |

หากไม่ได้กำหนดขีดจำกัด เครื่องมือจะแสดง "N/A" — การติดตามอายุเป็นตัวเลือก

## การมอบหมายเครื่องมือให้เครื่องจักร

เมื่อ Tool Instance ถูกมอบหมายให้เครื่องจักร:
- สถานะเปลี่ยนเป็น **In Use**
- บันทึกหมายเลข Magazine Pocket
- เครื่องจักรเพียงเครื่องเดียวเท่านั้นที่สามารถถือ Tool Instance ที่กำหนดได้ในเวลาเดียวกัน

การยกเลิกการมอบหมายเครื่องมือจะ Reset สถานะกลับเป็น **Available**

## Process Associations (ความสัมพันธ์กับกระบวนการ)

Tool Definitions สามารถเชื่อมโยงกับ Process Definitions — ระบุว่าเครื่องมือไหนใช้ในการดำเนินงานไหน ช่วยในการเตรียมเครื่องมือก่อนเริ่มงาน

## แนวคิดหลัก

| คำศัพท์ | ความหมาย |
|---------|----------|
| Tool Definition | เทมเพลต — ประเภทเครื่องมือและขีดจำกัดอายุ |
| Tool Instance | เครื่องมือจริง — ติดตามการใช้งานจริง |
| Life % | มิติที่ใช้มากที่สุดเป็นเปอร์เซ็นต์ของขีดจำกัด |
| Magazine pocket | หมายเลขช่องใน Tool Carousel ของเครื่องจักร |
| Process association | การดำเนินงานไหนใช้เครื่องมือประเภทไหน |
