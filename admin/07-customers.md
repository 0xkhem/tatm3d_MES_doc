# 7. Customers (ลูกค้า)

## การเข้าถึงลูกค้า

ไปที่ **Knowledge → Basic Data → Customer** หรือเข้าที่ `/knowledge/basic/customer`

[image: customer-list — ภาพหน้าจอเต็มของหน้ารายการลูกค้า]

---

## รายการลูกค้า

### คอลัมน์ในตาราง

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| Customer Code (รหัสลูกค้า) | รหัสเฉพาะ |
| Name (ชื่อ) | ชื่อบริษัท (เช่น "XPeng Motors", "Ford", "Bosch") |
| Contact Person (ผู้ติดต่อ) | ชื่อผู้ติดต่อหลัก |
| Phone (โทรศัพท์) | หมายเลขโทรศัพท์ |
| Email (อีเมล) | อีเมลผู้ติดต่อ |
| Status (สถานะ) | Active / Inactive |
| Actions (การดำเนินการ) | แก้ไข (✏), ลบ (🗑) |

---

## การสร้างลูกค้า

1. คลิก **+ Customer**
2. กรอก:

| ฟิลด์ | จำเป็น | คำอธิบาย |
|-------|--------|----------|
| Customer Code (รหัสลูกค้า) | ใช่ | รหัสเฉพาะ |
| Name (ชื่อ) | ใช่ | ชื่อบริษัท |
| Contact Person (ผู้ติดต่อ) | ไม่ | ผู้ติดต่อหลัก |
| Phone (โทรศัพท์) | ไม่ | หมายเลขโทรศัพท์ |
| Email (อีเมล) | ไม่ | ที่อยู่อีเมล |
| Delivery Address (ที่อยู่จัดส่ง) | ไม่ | ที่อยู่จัดส่ง (แสดงในใบเสนอราคาและใบส่งของ) |
| Invoice Address (ที่อยู่วางบิล) | ไม่ | ที่อยู่วางบิล (แสดงในใบเสนอราคา) |
| Tax ID (เลขประจำตัวผู้เสียภาษี) | ไม่ | เลขประจำตัวผู้เสียภาษี |
| Status (สถานะ) | ใช่ | Active หรือ Inactive |

3. คลิก **Save** (บันทึก)

## การใช้งานข้อมูลลูกค้า

- **Quotes** (ใบเสนอราคา) — ดรอปดาวน์เลือกลูกค้า
- **Sales Orders** (ใบสั่งขาย) — ลูกค้าเชื่อมโยงกับแต่ละคำสั่งซื้อ
- **Work Orders** (ใบสั่งผลิต) — ชื่อลูกค้าสืบทอดจาก SO
- **Quote PDF** (ไฟล์ PDF ใบเสนอราคา) — ที่อยู่ลูกค้าพิมพ์บนเอกสาร
- **Packing Slips** (ใบส่งของ) — ที่อยู่จัดส่งใช้สำหรับการขนส่ง
