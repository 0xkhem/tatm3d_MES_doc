# 8. Operators (พนักงานปฏิบัติการ)

## การเข้าถึง Operators

ไปที่ **Knowledge → Basic Data → Operator** หรือเข้าที่ `/knowledge/basic/operator`

[image: operator-list — ภาพหน้าจอเต็มของหน้ารายการพนักงานปฏิบัติการ]

---

## Operator คืออะไร?

พนักงานหน้างานที่ปฏิบัติงานกับเครื่องจักรและรายงานความก้าวหน้าของการผลิต พนักงานปฏิบัติการแต่ละคนมีรหัสเฉพาะ (OPR-XXXX) ที่ใช้เป็น PIN สำหรับหน้าจอปฏิบัติการบนมือถือ

---

## รายการ Operator

### คอลัมน์ในตาราง

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| Operator Code (รหัสพนักงาน) | รหัสเฉพาะ (เช่น `OPR-0042`) — นี่คือ PIN สำหรับเข้าสู่ระบบ |
| Name (ชื่อ) | ชื่อเต็มของพนักงานปฏิบัติการ |
| Status (สถานะ) | Active / Inactive |
| Skills (ทักษะ) | จำนวนทักษะที่กำหนด |
| Actions (การดำเนินการ) | แก้ไข (✏), ลบ (🗑) |

---

## การสร้าง Operator

1. คลิก **+ Operator**
2. กรอก:

| ฟิลด์ | จำเป็น | คำอธิบาย |
|-------|--------|----------|
| Operator Code (รหัสพนักงาน) | ใช่ | รูปแบบ: `OPR-XXXX` (4 หลัก) นี่คือ PIN สำหรับหน้าจอปฏิบัติการ |
| Name (ชื่อ) | ใช่ | ชื่อเต็ม |
| Name (Thai) (ชื่อภาษาไทย) | ไม่ | ชื่อภาษาไทย (แสดงในหน้าจอปฏิบัติการที่เป็นภาษาไทย) |
| Status (สถานะ) | ใช่ | Active หรือ Inactive เฉพาะพนักงานที่ Active เท่านั้นที่สามารถเข้าสู่ระบบได้ |

3. คลิก **Save** (บันทึก)

## แท็บทักษะ Operator (Operator Skills Tab)

พนักงานปฏิบัติการแต่ละคนสามารถมีการกำหนดทักษะที่เชื่อมโยงกับ Process Definition ที่ตนมีคุณสมบัติปฏิบัติงานได้:

1. เปิดหน้าแก้ไขของ Operator
2. ไปที่แท็บ **Skills** (ทักษะ)
3. เพิ่ม Process Definition ที่พนักงานได้รับการฝึกอบรม
4. ข้อมูลนี้ใช้สำหรับการวางแผนพนักงาน-เครื่องจักรและมุมมองข้อมูลหลัก

[image: operator-skills — ภาพหน้าจอแท็บ Skills ในหน้าแก้ไข Operator]

---

## การใช้งานข้อมูล Operator

- **Operator Home** (`/operator`) — เข้าสู่ระบบด้วย PIN รหัส 4 หลัก
- **Shift Roster** (ตารางกะ) — กำหนดให้เครื่องจักรตามกะ
- **Step Execution** (การปฏิบัติงานขั้นตอน) — บันทึกเป็นผู้ปฏิบัติงานเริ่มต้น/เสร็จสิ้นขั้นตอน WO
- **Abnormal Reports** (รายงานผิดปกติ) — เชื่อมโยงเป็นผู้รายงาน
- **Performance Tracking** (การติดตามประสิทธิภาพ) — ขั้นตอนที่เสร็จสิ้นต่อกะ
