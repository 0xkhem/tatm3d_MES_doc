# 1. Factory Template & Setup Wizard (เทมเพลตโรงงานและตัวช่วยตั้งค่า)

## ภาพรวม

Factory Template เป็นวิธีที่เร็วที่สุดในการเริ่มต้นใช้งาน TAM3D MES ระบบจะเติมข้อมูลหลักจากเทมเพลตเฉพาะอุตสาหกรรมให้อัตโนมัติ ช่วยประหยัดเวลาการกรอกข้อมูลด้วยตนเองหลายชั่วโมง

---

## การเข้าถึง Factory Template

ไปที่ **System → Factory Template** หรือเข้าที่ `/system/factory-template`

---

## เทมเพลตที่มี: CNC HMLV

ระบบมีเทมเพลตในตัวหนึ่งรายการที่ปรับแต่งสำหรับโรงงาน CNC แบบ High-Mix Low-Volume (ผลิตหลากหลายชิ้นงานจำนวนน้อย):

| หมวดหมู่ | รายการที่กำหนดค่าไว้ล่วงหน้า |
|----------|------------------------------|
| Equipment Types (ประเภทเครื่องจักร) | CNC Lathe, 3-Axis Mill, 5-Axis Mill, Swiss Lathe, Grinder, EDM, Bandsaw, CMM, Deburring Station, Packing Station |
| Equipment Manufacturers (ผู้ผลิตเครื่องจักร) | แบรนด์ CNC ที่นิยม |
| Maintenance Types (ประเภทการบำรุงรักษา) | Preventive (เชิงป้องกัน), Corrective (เชิงแก้ไข), Predictive (เชิงทำนาย) |
| Maintenance Strategies (กลยุทธ์การบำรุงรักษา) | ตารางรายวัน, รายสัปดาห์, รายเดือน |
| Abnormal Reasons (เหตุผลผิดปกติ) | หมวด Machine, Tooling, Material, Quality, Safety |
| Process Definitions (คำจำกัดความกระบวนการ) | Die Casting, Deburring, CNC Milling, CNC Turning, Surface Treatment, QC, Packing เป็นต้น |
| Process Categories (หมวดกระบวนการ) | Machining, Finishing, Inspection, Assembly |
| Material Categories (หมวดวัสดุ) | Aluminum, Steel, Stainless Steel, Brass, Titanium, Plastics |
| Shift Definitions (การกำหนดกะทำงาน) | กะกลางวัน, กะกลางคืน |
| Scheduling Rules (กฎการจัดตารางเวลา) | พารามิเตอร์เริ่มต้น |
| Factory Hierarchy (ลำดับชั้นโรงงาน) | 1 โรงงาน → 1 Workshop → 3 Production Line → 8 Workstation |

[image: factory-template-preview — ภาพหน้าจอแสดงตัวอย่างเทมเพลตพร้อมจำนวนรายการในแต่ละหมวดหมู่]

## การใช้เทมเพลต

### กรณีระบบยังไม่มีข้อมูล:

1. คลิก **Apply Template** (ใช้เทมเพลต)
2. Setup Wizard (ตัวช่วยตั้งค่า) จะเปิดขึ้น (ดูรายละเอียดด้านล่าง)

### กรณีระบบมีข้อมูลอยู่แล้ว:

1. คลิก **Apply Template** (ใช้เทมเพลต)
2. กล่องโต้ตอบคำเตือนจะปรากฏขึ้น: ข้อมูลที่มีอยู่จะถูกล้าง
3. พิมพ์ข้อความยืนยันเพื่อดำเนินการต่อ
4. คลิก **Confirm Reset** (ยืนยันการรีเซ็ต)
5. ข้อมูล Knowledge ที่มีอยู่ทั้งหมดจะถูกล้าง
6. Setup Wizard จะเปิดขึ้น

[image: factory-template-reset-dialog — ภาพหน้าจอกล่องโต้ตอบยืนยันการรีเซ็ตพร้อมข้อความเตือน]

> **คำเตือน:** การรีเซ็ตจะล้างข้อมูลหลักทั้งหมด (วัสดุ, เครื่องจักร, คำจำกัดความกระบวนการ เป็นต้น) การดำเนินการนี้ไม่สามารถย้อนกลับได้ ใช้เฉพาะกรณีเริ่มต้นใหม่เท่านั้น

## Setup Wizard (ตัวช่วยตั้งค่า 8 ขั้นตอน)

หลังจากใช้เทมเพลต ตัวช่วย 8 ขั้นตอนจะนำทางคุณผ่านการปรับแต่ง:

### ขั้นตอนที่ 1: ข้อมูลโรงงาน (Factory Info)
- ชื่อบริษัท, ที่อยู่, ข้อมูลติดต่อ
- ข้อมูลนี้จะแสดงในหัวเอกสารใบเสนอราคา

### ขั้นตอนที่ 2: ลำดับชั้นโรงงาน (Factory Hierarchy)
- ตรวจสอบและปรับแต่ง: Workshop, Production Line, Workstation
- เพิ่ม/ลบ/เปลี่ยนชื่อโหนดในแผนผังลำดับชั้น

### ขั้นตอนที่ 3: ประเภทเครื่องจักรและผู้ผลิต (Equipment Types & Manufacturers)
- ตรวจสอบประเภทเครื่องจักรที่เติมไว้ล่วงหน้า
- เพิ่มประเภทที่กำหนดเอง ลบประเภทที่ไม่ใช้
- กำหนดค่าผู้ผลิต

### ขั้นตอนที่ 4: ประเภทและกลยุทธ์การบำรุงรักษา (Maintenance Types & Strategies)
- ตรวจสอบคำจำกัดความประเภทการบำรุงรักษา
- กำหนดค่าตารางและกลยุทธ์การบำรุงรักษา

### ขั้นตอนที่ 5: เหตุผลผิดปกติและหมวดหมู่ (Abnormal Reasons & Categories)
- ตรวจสอบหมวดหมู่เหตุผลผิดปกติ (Machine, Tooling, Material, Quality, Safety)
- เพิ่ม/ลบเหตุผลเฉพาะในแต่ละหมวดหมู่

### ขั้นตอนที่ 6: คำจำกัดความและหมวดกระบวนการ (Process Definitions & Categories)
- ตรวจสอบคำจำกัดความกระบวนการที่เติมไว้ล่วงหน้า
- เพิ่มกระบวนการที่กำหนดเอง ตั้งค่าการเชื่อมโยงกับประเภทเครื่องจักร
- กำหนดค่าหมวดกระบวนการ

### ขั้นตอนที่ 7: กฎรูปแบบรหัส (Code Format Rules)
- กำหนดค่ารูปแบบรหัสอัตโนมัติสำหรับแต่ละประเภทข้อมูล
- ตั้งค่า Prefix (คำนำหน้า), Separator (ตัวคั่น), Year Format (รูปแบบปี), Sequence Length (ความยาวลำดับ)
- ดูตัวอย่างรหัสที่สร้าง (เช่น `WO-2026-00001`)

### ขั้นตอนที่ 8: เสร็จสิ้น (Completion)
- สรุปข้อมูลทั้งหมดที่กำหนดค่า
- คลิก **Finish** (เสร็จสิ้น) เพื่อบันทึกทุกอย่างลงฐานข้อมูล

[image: setup-wizard-step7 — ภาพหน้าจอขั้นตอนที่ 7 (Code Format) แสดงการกำหนดค่ารูปแบบรหัส]

## หลังจากตัวช่วยตั้งค่าเสร็จสิ้น

เมื่อตัวช่วยตั้งค่าเสร็จสมบูรณ์ ข้อมูลหลักทั้งหมดจะถูกเติม จากนั้นคุณสามารถ:
- ปรับแต่งข้อมูลแต่ละรายการในโมดูล Knowledge
- เพิ่มวัสดุ, ลูกค้า และพนักงานปฏิบัติการ (ซึ่งไม่ได้รวมอยู่ในเทมเพลต)
- กำหนดค่ากะทำงานและปฏิทินทำงานใน Schedule Modeling
- ตั้งค่าเลขลำดับอัตโนมัติใน System → Running Numbers
