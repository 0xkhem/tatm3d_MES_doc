## Factory Modeling คืออะไร?

การแสดงผลเชิงภาพของผังโรงงานจริง: Workshop → สายการผลิต → Workstation แต่ละ Workstation เชื่อมโยงกับเครื่องจักรจริง สร้างสะพานเชื่อมระหว่างโลกเชิงตรรกะ (ประเภทเครื่องจักร, คำจำกัดความกระบวนการ) กับโลกกายภาพ (เครื่องจักรจริงในหน้างาน)

---

## เค้าโครงหน้า (Page Layout)

### แถบสรุป (Summary Strip)
แถบ KPI ด้านบนแสดง:
- จำนวน Workstation ทั้งหมด
- เครื่องจักรที่ใช้งานอยู่
- เครื่องจักรที่อยู่ในสถานะ Hold
- เครื่องจักรที่อยู่ในการซ่อมบำรุง
- เปอร์เซ็นต์การใช้งาน

### การ์ด Workshop
แต่ละ Workshop แสดงเป็นการ์ดที่ประกอบด้วย:
- ชื่อ Workshop
- สายการผลิตภายใน Workshop
- Workstation Tile ภายในแต่ละสาย

### Workstation Tile
แต่ละ Tile แสดง:
- ชื่อและรหัสเครื่องจักร
- ตัวบ่งชี้สถานะ (เขียว=ใช้งาน, เหลือง=hold, แดง=ซ่อมบำรุง, เทา=ว่าง)
- ช่างปฏิบัติงานปัจจุบัน (จากตารางกะ)
- Work Order ปัจจุบัน (ถ้ามีขั้นตอนกำลังดำเนินการบนเครื่องนี้)

[image: factory-modeling-workshop — ภาพใกล้ของการ์ด Workshop แสดงสายการผลิตและ Workstation Tile พร้อมสีสถานะ]

## การแก้ไขโครงสร้างโรงงาน

1. คลิกไอคอน **Settings** (⚙) ในแถบเครื่องมือ
2. แผง **Factory Structure Editor** จะเปิดจากด้านขวา
3. คุณสามารถ:
   - เพิ่ม/เปลี่ยนชื่อ/ลบ Workshop
   - เพิ่ม/เปลี่ยนชื่อ/ลบสายการผลิตภายใน Workshop
   - เพิ่ม/เปลี่ยนชื่อ/ลบ Workstation ภายในสาย
   - เชื่อมโยง Workstation กับเครื่องจักร
4. การเปลี่ยนแปลงจะถูกบันทึกเมื่อปิด Editor

[image: factory-modeling-editor — ภาพหน้าจอของแผง Factory Structure Editor]

---

## แผ่น Workstation Detail

คลิก Workstation Tile ใดก็ได้เพื่อเปิดแผ่นรายละเอียดที่แสดง:
- ข้อมูลเครื่องจักร (ชื่อ, รหัส, ประเภท, ผู้ผลิต)
- สถานะปัจจุบันและช่างปฏิบัติงาน
- Work Order และขั้นตอนปัจจุบัน
- ประวัติการผลิตล่าสุด
- การกำหนดเครื่องมือ (ถ้ามีการตั้งค่า Tooling)

[image: factory-modeling-detail — ภาพหน้าจอของแผ่น Workstation Detail]

## ลำดับชั้นโรงงาน

```
โรงงาน (บริษัท)
  └── Workshop (เช่น "CNC Workshop")
        └── สายการผลิต (เช่น "5-Axis Line")
              └── Workstation (เช่น "Station 1" → เชื่อมกับ EQP-0001)
```

ลำดับชั้นนี้ถูกใช้โดย:
- การแสดงผล Factory Overview
- การติดตามตำแหน่งเครื่องจักร
- ฟีเจอร์การวางแผนกำลังการผลิตในอนาคต
