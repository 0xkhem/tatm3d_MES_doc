# 11. การจัดการเครื่องมือ (Tooling Management)

## การเข้าถึงหน้า Tooling

ไปที่ **Knowledge → Tooling** ในแถบด้านข้าง มีสามหน้าย่อย:

| หน้า | เส้นทาง | วัตถุประสงค์ |
|------|------|---------|
| Categories | `/knowledge/tooling/category` | จัดกลุ่มคำจำกัดความเครื่องมือ |
| Definitions | `/knowledge/tooling/definition` | เทมเพลตเครื่องมือที่ใช้ซ้ำได้พร้อมขีดจำกัดอายุการใช้งาน |
| Instances | `/knowledge/tooling/instance` | เครื่องมือจริงพร้อมการติดตามอายุการใช้งานปัจจุบัน |

## ลำดับชั้นของ Entity

```

                     ↕
            Process Definition (many-to-many)
```

## Tool Categories (หมวดหมู่เครื่องมือ)

การจัดกลุ่มสำหรับคำจำกัดความเครื่องมือ (เช่น "End Mills", "Drills", "Inserts", "Taps")

- รายการง่ายๆ พร้อมเพิ่ม/แก้ไข/ลบ
- แต่ละหมวดหมู่มีชื่อและคำอธิบายเพิ่มเติม

[image: tooling-categories — ภาพหน้าจอของหน้า Tool Categories]

## Tool Definitions (คำจำกัดความเครื่องมือ)

เทมเพลตเครื่องมือตัดที่ใช้ซ้ำได้ ซึ่งกำหนดว่าเครื่องมือคืออะไรและขีดจำกัดอายุการใช้งาน

### คอลัมน์ในตาราง

| คอลัมน์ | คำอธิบาย |
|--------|-------------|
| Code | สร้างอัตโนมัติ (เช่น `TLD-0001`) |
| Name | ชื่อเครื่องมือ (เช่น "Ø10 Carbide End Mill") |
| Category | หมวดหมู่เครื่องมือ |
| Max Life (Cuts) | จำนวนการตัดสูงสุดก่อนเปลี่ยน |
| Max Life (Minutes) | นาทีการกัดสูงสุด |
| Max Life (Parts) | จำนวนชิ้นงานสูงสุดที่ผลิตได้ |
| Status | Active / Inactive |

### การสร้าง Tool Definition

| ฟิลด์ | จำเป็น | คำอธิบาย |
|-------|----------|-------------|
| Code | ใช่ | สร้างอัตโนมัติ |
| Name | ใช่ | ชื่อที่อธิบายได้ |
| Category | ใช่ | เลือกจากหมวดหมู่เครื่องมือ |
| Max Life Cuts | ไม่ | ขีดจำกัดอายุตามจำนวนการตัด |
| Max Life Minutes | ไม่ | ขีดจำกัดอายุตามเวลากัด |
| Max Life Parts | ไม่ | ขีดจำกัดอายุตามจำนวนชิ้นงาน |
| Process Associations | ไม่ | คำจำกัดความกระบวนการที่ใช้เครื่องมือนี้ |

## Tool Instances (เครื่องมือจริง)

เครื่องมือจริง — เครื่องมือที่มีอยู่จริงในคลังเครื่องมือของคุณ

### คอลัมน์ในตาราง

| คอลัมน์ | คำอธิบาย |
|--------|-------------|
| Code | สร้างอัตโนมัติ (เช่น `TLI-0001`) |
| Definition | คำจำกัดความเครื่องมือที่เชื่อมโยง |
| Current Life (Cuts) | จำนวนการตัดปัจจุบัน |
| Current Life (Minutes) | นาทีการกัดปัจจุบัน |
| Current Life (Parts) | จำนวนชิ้นงานปัจจุบัน |
| Life % | เปอร์เซ็นต์อายุการใช้งานที่คำนวณ (สูงสุดในทุกมิติ) |
| Status | Available, In Use, Worn, Retired |

### สีสถานะอายุการใช้งาน

| Life % | สถานะ | สี |
|--------|--------|-------|
| < 80% | ปกติ | เขียว |
| 80–99% | เตือน | เหลือง |
| ≥ 100% | วิกฤต | แดง |
| ไม่มีขีดจำกัด | N/A | เทา |

## การกำหนดให้เครื่องจักร (Equipment Assignment)

Tool Instance สามารถกำหนดให้เครื่องจักรเฉพาะพร้อมตำแหน่งช่องแม็กกาซีน:

1. ในหน้า Tool Instance คลิก **Assign to Equipment**
2. เลือกเครื่องจักรและหมายเลขช่องแม็กกาซีน
3. สถานะ Instance เปลี่ยนเป็น "In Use"
4. การยกเลิกการกำหนดจะรีเซ็ตสถานะเป็น "Available"

> **หมายเหตุ:** Tool Instance หนึ่งชิ้นสามารถอยู่บนเครื่องจักรได้เพียงเครื่องเดียวในแต่ละครั้ง

---

## การแจ้งเตือนอายุเครื่องมือ (Tool Life Alerts)

เมื่อ Tool Instance เกินขีดจำกัดอายุการใช้งาน:
- **เตือน** (80–99%) — การแจ้งเตือนสีเหลืองในการแจ้งเตือนช่างปฏิบัติงาน
- **วิกฤต** (≥100%) — การแจ้งเตือนสีแดงในการแจ้งเตือนช่างปฏิบัติงานและรายการที่ต้องดำเนินการในหน้าหลัก

การแจ้งเตือนเหล่านี้ช่วยป้องกันการแตกหักของเครื่องมือและปัญหาคุณภาพจากเครื่องมือที่สึกหรอ
