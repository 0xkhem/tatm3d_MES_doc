# 2. Materials (วัสดุ)

## การเข้าถึงวัสดุ

ไปที่ **Knowledge → Basic Data → Material** หรือเข้าที่ `/knowledge/basic/material`

---

## หน้ารายการวัสดุ (Material List Page)

### แถบเครื่องมือ (Toolbar)
- ปุ่ม **+ Material** — สร้างวัสดุใหม่
- **Search box** (ช่องค้นหา) — กรองตามรหัสวัสดุหรือชื่อ

### คอลัมน์ในตาราง

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| Material Code (รหัสวัสดุ) | สร้างอัตโนมัติหรือกรอกเอง (เช่น `MAT-0001`) |
| Name (ชื่อ) | ชื่อวัสดุ (เช่น "Aluminum 6061-T6") |
| Type (ประเภท) | ประเภท/หมวดหมู่วัสดุ |
| Unit (หน่วย) | หน่วยวัด (เช่น "kg", "pcs", "m") |
| Grade (เกรด) | เกรดวัสดุ |
| Stock Form (รูปแบบสต็อก) | แท่ง, แผ่น, เพลท, เส้น เป็นต้น |
| Actions (การดำเนินการ) | แก้ไข (✏), ลบ (🗑) |

[image: materials-table — ภาพขยายของตารางวัสดุแสดง 4-5 แถวตัวอย่าง]

## การสร้างวัสดุ

1. คลิก **+ Material**
2. กรอกแบบฟอร์ม:

| ฟิลด์ | จำเป็น | คำอธิบาย |
|-------|--------|----------|
| Material Code (รหัสวัสดุ) | ใช่ | รหัสตามรูปแบบที่กำหนด (เช่น `MAT-0001`) |
| Name (ชื่อ) | ใช่ | ชื่อเต็มของวัสดุ (เช่น "Die Cast Aluminum Alloy A380") |
| Material Type (ประเภทวัสดุ) | ใช่ | หมวดหมู่ (เช่น "Aluminum", "Steel", "Stainless Steel") |
| Unit (หน่วย) | ใช่ | หน่วยวัด |
| Material Grade (เกรดวัสดุ) | ไม่ | ข้อมูลจำเพาะเกรด (เช่น "T6", "4140") |
| Stock Form (รูปแบบสต็อก) | ไม่ | รูปแบบทางกายภาพ (เช่น "Bar", "Sheet", "Plate") |
| Supplier (ผู้จัดหา) | ไม่ | ชื่อผู้จัดหาเริ่มต้น |
| Standard Cost (ต้นทุนมาตรฐาน) | ไม่ | ต้นทุนต่อหน่วย |
| Description (คำอธิบาย) | ไม่ | หมายเหตุเพิ่มเติม (สูงสุด 500 ตัวอักษร) |

3. คลิก **Save** (บันทึก)

[image: material-form — ภาพหน้าจอแบบฟอร์มสร้างวัสดุพร้อมข้อมูลตัวอย่าง]

---

## การแก้ไขวัสดุ

1. คลิก **ไอคอนดินสอ** (✏) ที่แถวของวัสดุ
2. แบบฟอร์มแก้ไขจะโหลดข้อมูลที่มีอยู่
3. ทำการเปลี่ยนแปลงและคลิก **Save** (บันทึก)

---

## การลบวัสดุ

1. คลิก **ไอคอนถังขยะ** (🗑) ที่แถวของวัสดุ
2. ยืนยันในกล่องโต้ตอบ

> **หมายเหตุ:** วัสดุที่ถูกอ้างอิงโดยใบสั่งผลิตหรือบันทึกอื่นจะไม่สามารถลบได้ ข้อความแสดงข้อผิดพลาดจะปรากฏขึ้น
