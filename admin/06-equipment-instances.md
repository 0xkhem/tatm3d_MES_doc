# 6. Equipment Instances (เครื่องจักรรายตัว)

## การเข้าถึงเครื่องจักร

ไปที่ **Knowledge → Equipment Modeling → Equipment Manufacturing** หรือเข้าที่ `/knowledge/equipment/manufacturing`

[image: equipment-list — ภาพหน้าจอเต็มของหน้ารายการเครื่องจักร]

---

## Equipment Instance คืออะไร?

เครื่องจักรทางกายภาพแต่ละเครื่องบนพื้นที่โรงงาน (เช่น "CNC-5A-01" ซึ่งเป็น 5-Axis CNC Mill เครื่องหนึ่ง) แต่ละเครื่องจักรจะสังกัด Equipment Type และสามารถกำหนดให้กับ Workstation, กะทำงาน และการดำเนินงานตามตารางเวลาได้

---

## รายการเครื่องจักร

### คอลัมน์ในตาราง

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| Equipment Code (รหัสเครื่องจักร) | รหัสเฉพาะ (เช่น `EQP-0001`) |
| Name (ชื่อ) | ชื่อเครื่องจักร (เช่น "Mazak Integrex i-200") |
| Equipment Type (ประเภทเครื่องจักร) | ประเภทที่เชื่อมโยง (เช่น "5-Axis CNC Mill") |
| Status (สถานะ) | Active, Inactive, Maintenance |
| Location (ตำแหน่ง) | การกำหนด Workshop/Line |
| Actions (การดำเนินการ) | แก้ไข (✏), ลบ (🗑) |

---

## การสร้างเครื่องจักร

1. คลิก **+ Equipment**
2. กรอก:

| ฟิลด์ | จำเป็น | คำอธิบาย |
|-------|--------|----------|
| Equipment Code (รหัสเครื่องจักร) | ใช่ | รหัสเฉพาะ (เช่น `EQP-0001`) |
| Name (ชื่อ) | ใช่ | ชื่อเครื่องจักร |
| Equipment Type (ประเภทเครื่องจักร) | ใช่ | ดรอปดาวน์ — เลือกจากประเภทเครื่องจักรที่กำหนดไว้ |
| Manufacturer (ผู้ผลิต) | ไม่ | ผู้ผลิตเครื่องจักร |
| Model (รุ่น) | ไม่ | หมายเลขรุ่น |
| Serial Number (หมายเลขซีเรียล) | ไม่ | หมายเลขซีเรียล |
| Status (สถานะ) | ใช่ | Active, Inactive หรือ Maintenance |
| Location (ตำแหน่ง) | ไม่ | คำอธิบายตำแหน่งทางกายภาพ |

3. คลิก **Save** (บันทึก)

## สถานะเครื่องจักร

| สถานะ | ความหมาย | ผลกระทบ |
|--------|---------|---------|
| Active (ใช้งาน) | เครื่องจักรพร้อมทำงาน | แสดงในบอร์ดจัดตารางเวลา สามารถกำหนดงานได้ |
| Inactive (ไม่ใช้งาน) | เครื่องจักรไม่ได้ใช้งาน | ซ่อนจากตารางเวลา |
| Maintenance (บำรุงรักษา) | เครื่องจักรอยู่ระหว่างการบำรุงรักษา | แสดงคำเตือนการบำรุงรักษาต่อผู้ปฏิบัติงาน ถูกทำเครื่องหมายในรายการแจ้งเตือน |

---

## ความสัมพันธ์กับข้อมูลอื่น

- **Equipment Type** — ทุกเครื่องจักรต้องสังกัดประเภทใดประเภทหนึ่ง
- **Scheduling Board** (บอร์ดจัดตารางเวลา) — เครื่องจักรที่ Active จะแสดงเป็นแถวในตาราง Timeline
- **Shift Roster** (ตารางกะ) — พนักงานปฏิบัติการถูกกำหนดให้เครื่องจักรเฉพาะตามกะ
- **Tool Assignments** (การกำหนดเครื่องมือ) — เครื่องมือตัดสามารถกำหนดให้เครื่องจักรพร้อมตำแหน่ง Magazine Pocket
- **Factory Modeling** (การจำลองโรงงาน) — เครื่องจักรสามารถเชื่อมโยงกับ Workstation ในลำดับชั้นโรงงาน
