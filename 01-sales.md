# Sales (การขาย)

## คืออะไร?

โมดูล Sales คือจุดเริ่มต้นของทุกคำสั่งซื้อ ครอบคลุมขั้นตอนก่อนการผลิตทั้งหมด ตั้งแต่การสร้างใบเสนอราคา การได้รับการอนุมัติจากลูกค้า ไปจนถึงการแปลงเป็น Sales Order ที่จะสร้าง Work Order ต่อไป

## ขั้นตอนการทำงาน

```
Quote (ใบเสนอราคา) → ลูกค้าอนุมัติ → Sales Order → Line Items → Work Orders
```

## Quote (ใบเสนอราคา)

Quote คือข้อเสนอราคาที่ส่งให้ลูกค้า สามารถ:

- เพิ่ม Line Items พร้อมชื่อชิ้นส่วน จำนวน ราคาต่อหน่วย และระยะเวลานำ
- แนบแบบหรือไฟล์อ้างอิง
- Export เป็น PDF สำหรับส่งภายนอก
- แชร์ผ่านลิงก์สาธารณะ — ลูกค้าดูได้ในเบราว์เซอร์โดยไม่ต้องล็อกอิน

เมื่อลูกค้ายืนยัน สามารถแปลง Quote เป็น Sales Order ได้ในคลิกเดียว

## Sales Order (คำสั่งซื้อ)

Sales Order (SO) คือการยืนยันคำสั่งซื้อจากลูกค้า แต่ละ SO มี Line Items — หนึ่งรายการต่อหมายเลขชิ้นส่วนและจำนวน จาก SO สามารถสร้าง Work Order สำหรับทุก Line Item พร้อมกันได้ผ่าน Flow Engine

รูปแบบรหัส SO: `SO-2026-001`

## Delivery Promise Signal (สัญญาณการส่งมอบ)

ทุก Sales Order แสดง Delivery Promise Signal — ตัวบ่งชี้สีเขียว/เหลือง/แดงที่ตรวจสอบว่าโรงงานมีกำลังการผลิตเพียงพอที่จะส่งมอบตามวันที่ขอหรือไม่ อัปเดตแบบ real-time เมื่อตารางการผลิตเปลี่ยนแปลง

- เขียว — เป็นไปตามแผน
- เหลือง — มีความเสี่ยง กำลังการผลิตตึงตัว
- แดง — ไม่สามารถส่งมอบตามวันที่กำหนดได้ด้วยภาระงานปัจจุบัน

## แนวคิดหลัก

| คำศัพท์ | ความหมาย |
|---------|----------|
| Quote | ใบเสนอราคา แชร์ให้ลูกค้าได้ |
| Sales Order | คำสั่งซื้อที่ยืนยันแล้วพร้อม Line Items |
| Line Item | ชิ้นส่วนหนึ่งรายการ + จำนวนใน SO |
| Delivery Promise | การตรวจสอบกำลังการผลิตแบบ real-time เทียบกับวันส่งมอบ |
